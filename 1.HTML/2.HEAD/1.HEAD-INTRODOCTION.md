
## ¿Qué es el `<head>`?

Es una sección del documento HTML que contiene **metadatos**: información sobre el documento en sí, y no sobre su contenido visible.  
Todo lo que se pone en `<head>` le indica al navegador **cómo debe interpretar, mostrar y procesar** la página.

Se escribe así:

  <!-- contenido -->

## LISTA DE ETIQUETAS USADAS MAS FRECUENTES:

1- **`<meta charset="UTF-8">`**  
    Define el tipo de codificación de caracteres (UTF-8 es el estándar).
    
2- **`<meta name="viewport" ...>`**  
    Hace que el sitio sea responsive y se vea bien en celulares.
    
3- **`<title>`**  
    Establece el título que aparece en la pestaña del navegador.
    
4- **`<link rel="stylesheet" href="...">`**  
    Conecta el archivo CSS externo con la página.
    
5- **`<link rel="icon" href="...">`**  
    Muestra el ícono (favicon) en la pestaña del navegador.
    
6- **`<style>`**  
    Permite escribir CSS directamente dentro del HTML (para estilos rápidos o internos).
    
7- **`<script src="...">`**  
    Enlaza un archivo JavaScript externo (se recomienda ubicarlo al final del body).
    
8- **`<meta ...>` (varios tipos)**  
    Metadatos para descripción, autor, palabras clave, redes sociales, SEO, compatibilidad, etc.
    
9- **`<link rel="preconnect">`, `<link rel="preload">`, etc.**  
    Optimización de carga anticipada de recursos (opcional, pero útil).
    
10- **`<base href="...">`**  
    Define la URL base para que los enlaces relativos funcionen correctamente.
