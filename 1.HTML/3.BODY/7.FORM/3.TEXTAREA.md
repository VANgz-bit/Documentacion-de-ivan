## **1. Introducción a `<textarea>`**

La etiqueta `<textarea>` en HTML se utiliza para **crear campos de texto multilínea** dentro de un formulario. A diferencia de `<input type="text">`, que está limitado a una sola línea, `<textarea>` permite al usuario **escribir párrafos, oraciones extensas o bloques de texto más grandes**, lo que lo hace ideal para comentarios, mensajes, descripciones o formularios de contacto.

La etiqueta es de tipo **contenedora**, es decir, tiene **apertura `<textarea>` y cierre `</textarea>`**, y el texto que coloquemos entre ambas etiquetas se mostrará como **contenido inicial o predeterminado**.

---

## **2. Sintaxis básica**

```html
<textarea name="comentario"></textarea>
```

En su forma más simple, esto generará un área rectangular de texto donde el usuario puede escribir libremente. Sin embargo, su **comportamiento y estilo** pueden personalizarse mediante atributos HTML y CSS.

---

## **3. Principales atributos de `<textarea>`**

A continuación, detallamos los atributos más importantes, con explicaciones claras y ejemplos:

|**Atributo**|**Descripción detallada**|**Valor por defecto**|**Ejemplo de uso**|
|---|---|---|---|
|**`name`**|Define el nombre del campo, que es clave al enviar datos a un servidor. El valor aquí se usará como clave en el backend.|—|`<textarea name="comentario"></textarea>`|
|**`id`**|Asigna un identificador único para enlazar con etiquetas `<label>` o con scripts de JavaScript.|—|`<textarea id="mensaje"></textarea>`|
|**`rows`**|Establece el número de **filas visibles** (líneas de texto) que mostrará el área inicialmente. No limita la cantidad total de texto.|—|`<textarea rows="6"></textarea>`|
|**`cols`**|Define el número de **caracteres visibles por línea**, calculando el ancho del área según la fuente.|—|`<textarea cols="40"></textarea>`|
|**`placeholder`**|Muestra un texto guía que desaparece cuando el usuario comienza a escribir. Útil como indicación.|—|`<textarea placeholder="Escribe tu comentario aquí..."></textarea>`|
|**`required`**|Indica que el campo **no puede enviarse vacío** al servidor. Si no se completa, el formulario no se envía.|`false`|`<textarea required></textarea>`|
|**`readonly`**|Convierte el área en **solo lectura**. Se muestra el contenido, pero no se puede editar.|`false`|`<textarea readonly>Texto fijo</textarea>`|
|**`disabled`**|Desactiva el campo. El usuario no puede interactuar y su valor no se envía al servidor.|`false`|`<textarea disabled>Campo deshabilitado</textarea>`|
|**`maxlength`**|Limita el **número máximo de caracteres** que el usuario puede escribir.|Ilimitado|`<textarea maxlength="250"></textarea>`|
|**`wrap`**|Controla cómo se ajusta el texto al llegar al borde.|`soft`|Ver explicación abajo.|

---

### **3.1. El atributo `wrap`**

Este atributo controla cómo se **ajusta el texto al enviarse al servidor**:

|**Valor**|**Comportamiento**|
|---|---|
|**`soft`** _(por defecto)_|El texto **no se ajusta visualmente** en el envío. Se envía exactamente como el usuario lo escribió, aunque en el área se vea ajustado al ancho.|
|**`hard`**|Inserta saltos de línea `\n` al final de cada línea visual en el área de texto. Así, lo que ves es exactamente lo que se envía.|

**Ejemplo:**

```html
<textarea name="comentario" wrap="hard" rows="5" cols="40"></textarea>
```

---

## **4. Texto por defecto**

Para agregar un **valor inicial** a un `<textarea>`, no se usa el atributo `value` (como en `<input>`).  
En su lugar, se coloca el texto **entre las etiquetas de apertura y cierre**:

**Ejemplo:**

```html
<textarea name="mensaje" rows="5" cols="40">
Este es un mensaje predeterminado.
</textarea>
```

---

## **5. Personalización con CSS**

El área de texto se puede **adaptar visualmente** con CSS. Algunas propiedades comunes son:

|**Propiedad**|**Función**|
|---|---|
|`width`, `height`|Definen el tamaño del área manualmente.|
|`resize`|Controla si el usuario puede redimensionar el campo (`none`, `both`, `horizontal`, `vertical`).|
|`padding`|Ajusta el espacio interno entre el borde y el texto.|
|`font-family`, `font-size`|Cambian la tipografía y el tamaño del texto.|
|`color`, `background-color`|Ajustan los colores de texto y fondo.|
|`border`, `border-radius`|Cambian el borde y redondean esquinas.|

**Ejemplo básico de estilo:**

```css
textarea {
  width: 100%;
  height: 150px;
  padding: 10px;
  font-size: 16px;
  font-family: Arial, sans-serif;
  border: 1px solid #ccc;
  border-radius: 8px;
  resize: none; /* Evita que el usuario cambie el tamaño */
}
```

---

## **6. Propiedad CSS moderna: `field-sizing`**

Esta propiedad, soportada en navegadores modernos, permite que el área de texto **cambie su tamaño automáticamente** a medida que el usuario escribe.

|**Valor**|**Efecto**|
|---|---|
|`fixed` _(por defecto)_|El área es fija, si no cabe el texto, aparece scroll.|
|`content`|El área se expande dinámicamente al escribir.|

**Ejemplo:**

```css
textarea {
  field-sizing: content;
}
```

---

## **7. Librerías para mejorar `<textarea>`**

Cuando necesitamos un área de texto más avanzada, podemos usar **editores enriquecidos (WYSIWYG)**. Estos permiten formato en negrita, listas, inserción de imágenes o edición colaborativa.

Algunas opciones:

- **Ligeros**: Quill, Medium Editor, Trix, Jodit.
    
- **Más completos**: TinyMCE, CKEditor, ProseMirror, Slate, AlloyEditor.
    

---

## **8. Ejemplo completo**

```html
<form action="/enviar" method="post">
  <label for="mensaje">Mensaje:</label>
  <textarea 
    id="mensaje"
    name="mensaje"
    placeholder="Escribe tu mensaje..."
    rows="6"
    cols="50"
    maxlength="250"
    required
  >Texto predeterminado.</textarea>
  <button type="submit">Enviar</button>
</form>
```
