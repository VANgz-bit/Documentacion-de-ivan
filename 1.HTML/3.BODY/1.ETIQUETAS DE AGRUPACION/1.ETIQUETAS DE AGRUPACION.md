Existen muchas **etiquetas HTML** dentro del lenguaje HTML5. Las etiquetas existentes se pueden desglosar en dos grupos principales:

- Las etiquetas que contienen **fragmentos de texto** (_dan significado o contexto a esos fragmentos_)
- Las etiquetas que **agrupan otras etiquetas** (_contienen y agrupan información variada_)

En esta ocasión, vamos a centrarnos en el **segundo grupo**, que son las etiquetas que se utilizan para agrupar y asociar información relacionada. Más adelante, profundizaremos en el resto de etiquetas.

Veamos un pequeño ejemplo de lo que es una **etiqueta contenedora**:

```html
<div class="container">
  <p>Hola, esto es un primer párrafo de ejemplo.</p>
  <p>Y esto, es un segundo párrafo de ejemplo.</p>
</div>
```

La etiqueta `<div class="container">` es lo que se llama un **contenedor**, es decir, una etiqueta que contiene un grupo de etiquetas relacionadas de alguna forma. Aunque no tiene que tener una clase de forma obligatoria, a las etiquetas `<div>` se les suele añadir una clase por varios motivos:

- Identificarla como una sección específica
- Para darle estilo visual (CSS)
- Para darle funcionalidad (Javascript)

> En muchos casos, una buena práctica de nombrado es añadirle el sufijo `-container` a la clase, de modo que si tenemos una clase `text`, el `<div>` que la contiene se llamará `text-container`.

### [Etiquetas HTML de agrupación](https://lenguajehtml.com/html/agrupacion/que-son/#etiquetas-html-de-agrupaci%C3%B3n)

Las siguientes etiquetas se utilizan para agrupar información, como es el caso de las etiquetas `<div>` o `<p>` que vimos en el ejemplo mostrado anteriormente:

|Etiqueta|Descripción|URL|
|---|---|---|
|`<div>`|Capa o división utilizado para agrupar varias etiquetas HTML.|[La etiqueta `<div>`](https://lenguajehtml.com/html/agrupacion/etiqueta-html-div/)|
|`<p>`|Define un párrafo de texto (_con sus etiquetas HTML para texto_).|[La etiqueta `<p>`](https://lenguajehtml.com/html/agrupacion/que-son/#p%C3%A1rrafo-de-texto-etiqueta-p)|
|`<main>`|Contenedor para englobar la parte principal de la página.|[La etiqueta `<main>`](https://lenguajehtml.com/html/semantica/etiqueta-html-section-main-aside/#la-etiqueta-main)|
|`<hr>`|Indica una separación temática del texto.|[La etiqueta `<hr>`](https://lenguajehtml.com/html/agrupacion/que-son/#separaci%C3%B3n-tem%C3%A1tica-etiqueta-hr)|
|`<ol>`|Crea una lista numerada (con orden).|[La etiqueta `<ol>`](https://lenguajehtml.com/html/agrupacion/etiquetas-html-ul-ol/)|
|`<ul>`|Crea una lista donde el orden no importa.|[La etiqueta `<ul>`](https://lenguajehtml.com/html/agrupacion/etiquetas-html-ul-ol/)|
|`<li>`|Contiene uno de los ítems de una lista numerada o sin numerar.|[La etiqueta `<li>`](https://lenguajehtml.com/html/agrupacion/etiquetas-html-ul-ol/)|
|`<pre>`|Establece un texto preformateado (_respetando espacios y saltos de línea_).|[La etiqueta `<pre>`](https://lenguajehtml.com/html/agrupacion/etiqueta-html-pre/)|
|`<blockquote>`|Agrupa información y características de una cita (_autor, fuente, etc..._).|[La etiqueta `<blockquote>`](https://lenguajehtml.com/html/agrupacion/etiqueta-html-blockquote/)|
|`<dl>`|Crea una lista de definiciones.|[La etiqueta `<dl>`](https://lenguajehtml.com/html/agrupacion/etiqueta-html-dl/)|
|`<dt>`|Establece el término de una definición.|[La etiqueta `<dt>`](https://lenguajehtml.com/html/agrupacion/etiqueta-html-dl/)|
|`<dd>`|Establece la descripción de un término de una definición.|[La etiqueta `<dd>`](https://lenguajehtml.com/html/agrupacion/etiqueta-html-dl/)|
|`<figure>`|Agrupa un elemento visual en una figura o ilustración.|[La etiqueta `<figure>`](https://lenguajehtml.com/html/agrupacion/etiqueta-html-figure/)|
|`<figcaption>`|Establece una leyenda a una figura o ilustración.|[La etiqueta `<figcaption>`](https://lenguajehtml.com/html/agrupacion/etiqueta-html-figure/)|
