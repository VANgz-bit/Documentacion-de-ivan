## **1. `<table>`**

### **Definición**

La etiqueta `<table>` es el **contenedor principal** de una tabla en HTML.  
Dentro de ella se ubican todas las filas, celdas y demás elementos que conforman la estructura de la tabla.

### **Características principales**

- No se puede utilizar fuera del flujo del documento HTML.
    
- Contiene a las etiquetas `<tr>`, `<th>` y `<td>`, entre otras más avanzadas como `<thead>`, `<tbody>` y `<tfoot>`.
    
- No tiene atributos obligatorios, aunque acepta atributos globales como `class`, `id` o `style`.
    

### **Ejemplo**

```html
<table>
  <tr>
    <td>Celda 1</td>
    <td>Celda 2</td>
  </tr>
</table>
```

---

## **2. `<tr>`**

### **Definición**

La etiqueta `<tr>` significa **Table Row** y se utiliza para definir una **fila** dentro de la tabla.

### **Características principales**

- Es un **elemento contenedor** que alberga celdas, las cuales pueden ser `<td>` (celdas de datos) o `<th>` (celdas de encabezado).
    
- No puede existir fuera de `<table>`, `<thead>`, `<tbody>` o `<tfoot>`.
    

### **Ejemplo**

```html
<table>
  <tr>
    <td>Fila 1 - Celda 1</td>
    <td>Fila 1 - Celda 2</td>
  </tr>
</table>
```

---

## **3. `<td>`**

### **Definición**

La etiqueta `<td>` significa **Table Data** y define una **celda de datos** en la tabla.

### **Características principales**

- Se coloca siempre dentro de una fila (`<tr>`).
    
- Puede contener texto, imágenes, enlaces u otros elementos HTML.
    
- Se puede personalizar con CSS para controlar ancho, alto, tipografía, etc.
    

### **Ejemplo**

```html
<table>
  <tr>
    <td>Nombre</td>
    <td>Edad</td>
  </tr>
  <tr>
    <td>Iván</td>
    <td>21</td>
  </tr>
</table>
```

---

## **4. `<th>`**

### **Definición**

La etiqueta `<th>` significa **Table Header** y se usa para crear una **celda de encabezado** en una tabla.

### **Características principales**

- Generalmente se usa en la primera fila o primera columna para **indicar títulos**.
    
- El texto dentro de `<th>` se muestra por defecto en **negrita** y **centrado**.
    
- Mejora la **semántica y accesibilidad**, ya que los lectores de pantalla identifican estas celdas como encabezados.
    

### **Ejemplo**

```html
<table>
  <tr>
    <th>Producto</th>
    <th>Precio</th>
  </tr>
  <tr>
    <td>Teclado</td>
    <td>$20</td>
  </tr>
</table>
```

## **5. Ejemplo completo**

Aquí un ejemplo de una tabla sencilla con encabezados y filas de datos:

```html
<table>
  <!-- Encabezados -->
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
    <th>País</th>
  </tr>
  <!-- Primera fila -->
  <tr>
    <td>Iván</td>
    <td>21</td>
    <td>Argentina</td>
  </tr>
  <!-- Segunda fila -->
  <tr>
    <td>Laura</td>
    <td>25</td>
    <td>México</td>
  </tr>
</table>
```
