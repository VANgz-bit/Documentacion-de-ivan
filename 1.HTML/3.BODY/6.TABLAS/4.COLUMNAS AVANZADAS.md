## **Tablas HTML: trabajar con columnas (`<colgroup>` y `<col>`)**

Cuando creamos una tabla, recordemos que la estructura es **horizontal**, usando `<tr>` para las filas. Esto significa que **los datos se organizan por filas**.  
Si queremos aplicar cambios o estilos **a una columna específica**, puede resultar más complicado, porque la columna no es un contenedor directo en HTML, sino que cada celda pertenece a su fila.

Esto puede hacerse con CSS, pero:

- Puede ser **difícil para quienes no dominan CSS avanzado**.
    
- Si queremos aplicar **información semántica** a una columna, CSS por sí solo no ayuda.
    

Para solucionar esto, HTML incluye **etiquetas específicas para trabajar con columnas**: `<colgroup>` y `<col>`.

---

### **1. La etiqueta `<col>`**

#### **Definición**

`<col>` representa **una columna dentro de una tabla**. No contiene celdas ni filas, pero permite **aplicar atributos, estilos o clases** a todas las celdas de esa columna.

#### **Características**

- Se coloca **dentro de `<colgroup>`**, que es su contenedor.
    
- Permite asignar clases o estilos a toda la columna, evitando tener que modificar cada `<td>` individualmente.
    
- Puede usar el atributo `span` para abarcar varias columnas contiguas, aplicando el mismo estilo o comportamiento.
    

---

### **2. La etiqueta `<colgroup>`**

#### **Definición**

`<colgroup>` es un **contenedor de columnas** que permite agrupar varias columnas y aplicarles comportamientos comunes o estilos compartidos.

#### **Reglas importantes**

- Debe aparecer dentro de `<table>`.
    
- Nunca debe colocarse **antes de `<caption>`** ni después de `<thead>`, `<tbody>` o `<tfoot>` si existen.
    
- Si usamos el atributo `span` en `<colgroup>`, no debemos incluir etiquetas `<col>` dentro.
    

---

### **3. Ejemplo práctico: colorear columnas con `<colgroup>` y `<col>`**

Supongamos que queremos que **cada columna de calificaciones tenga un color diferente** para mejorar la lectura:

```html
<table border="1">
  <caption>Calificaciones de los alumnos</caption>
  
  <colgroup>
    <col> <!-- Columna Estudiante sin color -->
    <col class="first"> <!-- Columna 1º ev. -->
    <col class="second"> <!-- Columna 2º ev. -->
    <col class="third"> <!-- Columna 3º ev. -->
  </colgroup>
  
  <thead>
    <tr class="title">
      <th rowspan="2">Estudiante</th>
      <th colspan="3">Calificaciones</th>
    </tr>
    <tr>
      <th>1º ev.</th>
      <th>2º ev.</th>
      <th>3º ev.</th>
    </tr>
  </thead>
  
  <tbody>
    <tr>
      <td>Bernardo</td>
      <td>5.7</td>
      <td>8.3</td>
      <td>8.9</td>
    </tr>
    <tr>
      <td>Remigio</td>
      <td>4.7</td>
      <td>4.7</td>
      <td>1.3</td>
    </tr>
    <tr>
      <td>Ana</td>
      <td>9.7</td>
      <td>9.7</td>
      <td>9.9</td>
    </tr>
  </tbody>
</table>
```

#### **CSS para colorear las columnas**

```css
.first { background-color: #ffcccc; }
.second { background-color: #ff9999; }
.third { background-color: #ff6666; }
```

> Observa que, usando `<col>` con clases, podemos **colorear todas las celdas de la columna** sin tener que modificar cada `<td>` individualmente.

---

### **4. Uso del atributo `span` en `<col>`**

`span` permite que **una columna aplique sus estilos a varias columnas contiguas**. Por ejemplo:

```html
<col span="2" class="highlight">
```

- Esto indica que las **dos primeras columnas** recibirán la clase `highlight`.
    
- Evita duplicar `<col>` para cada columna y facilita el mantenimiento.
    

---

### **5. Alternativa con CSS puro**

Podemos aplicar colores a columnas sin `<col>` usando selectores CSS avanzados:

```css
tbody td:nth-child(2) { background: #ff9999; }
tbody td:nth-child(3) { background: #ff5555; }
tbody td:nth-child(4) { background: #ff2222; }
```

**Limitaciones:**

- Solo afecta las **celdas del `<tbody>`** (no las cabeceras `<th>`).
    
- Puede complicarse si la tabla tiene muchas filas o columnas.
    
- Menos semántico, porque no define directamente la columna como un elemento.
    

---

### **6. Resumen de `<colgroup>` y `<col>`**

|Etiqueta|Función principal|
|---|---|
|`<colgroup>`|Agrupa columnas para aplicar estilos o comportamientos a varias columnas a la vez.|
|`<col>`|Representa una columna individual y permite asignar estilos, clases o atributos. Puede usar `span` para abarcar varias columnas.|

**Beneficios:**

- Permite aplicar estilos **columnas completas** de forma sencilla.
    
- Mejora la **semántica** de la tabla.
    
- Facilita la **accesibilidad** y el mantenimiento del código.
