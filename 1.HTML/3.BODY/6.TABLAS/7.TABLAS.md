las tablas son elementos que usamos para organizar información de forma clara y organizar datos en columnas si es necesario, para ello usamos estas etiquetas, la primera y principal es la que define lo que vamos a crear <`table`> como dije las tablas tienen varias etiquetas dentro tenemos 3 que son las mas importantes:

<`th`> (table header)son los encabezados de una columna.
<`tr`> (tablerows) se encarga de crear una fila y dentro de ella tenemos:
<`td`> se encarga de crear celdas de datos normal para las filas.

Para que google entienda que la primer fila es el encabezado usamos la etiqueta <`thead`>
de esta forma tenemos mejor organizado los titulares; Y también le damos un <`tbody`> para decirle que dentro de el esta los datos de la tabla; Luego esta el <`tfoot`> que viene siendo como el pie de la tabla y resume toda la información.

DE TAL FORMA QUE NOS QUEDARIA ASI

`<body>`
    `<table>`
        `<thead>`
            `<tr>`
                 `<td>PRODUCTO</td>`
	         `<td>PRECIO</td>`
                `<td>VENTAS</td>`
            `</tr>`
        `</thead>`
        `<tbody>`
            `<tr>`
                `<td>TECLADO</td>`
                `<td>10DLS</td>`
                `<td>10 UNIDADES</td>`
            `</tr>`
                `<td>MOUSE</td>`
                `<td>5DLS</td>`

            `<tr>`
                `<td>MICRO</td>`
                `<td>20DLS</td>`
                `<td>11UNIDAES</td>`
                `</tr>`
        `</tbody>`
    `</table>`
    `<tfoot>`
        `<th>TOTAL</th>`
            `<tr>`
                `<td colspan="2">35DLS</td>`
            `</tr>`
