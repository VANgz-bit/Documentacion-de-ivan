## 🔹 Etiquetas `<sup>` y `<sub>` en HTML

En el desarrollo de documentos web, especialmente cuando trabajamos con textos científicos, matemáticos, químicos o simplemente queremos representar ciertos formatos de escritura especializada, **los superíndices y subíndices son herramientas fundamentales**. HTML proporciona dos etiquetas específicas para esto: `<sup>` y `<sub>`.

---

### ✅ ¿Qué es un superíndice?

Un **superíndice** es un carácter (letra, número o símbolo) que se representa visualmente **en una posición superior al resto del texto**. Suele utilizarse en:

- Exponentes matemáticos: `10²`, `x³`
    
- Referencias a notas al pie: `Texto¹`
    
- Variables con exponentes: `xⁿ`
    

### ✅ ¿Qué es un subíndice?

Un **subíndice**, por el contrario, es un carácter que se representa **en una posición inferior al texto estándar**. Se usa comúnmente en:

- Fórmulas químicas: `H₂O`, `CO₂`
    
- Variables con índices: `x₁`, `x₂`
    
- Expresiones técnicas: `log₂(x)`
    

---

## 🔸 La etiqueta `<sup>`

La etiqueta `<sup>` (**superscript**) sirve para marcar contenido que debe aparecer como superíndice.

### ✔️ Sintaxis:

```html
<sup>contenido</sup>
```

### ✔️ Ejemplo:

```html
<p>El resultado de 24<sup>2</sup> es <strong>576</strong>.</p>
```

**Resultado visual esperado:**

> El resultado de 24² es **576**.

Esto es útil para representar potencias, como en matemáticas o física.

---

## 🔸 La etiqueta `<sub>`

La etiqueta `<sub>` (**subscript**) se utiliza para marcar contenido como subíndice.

### ✔️ Sintaxis:

```html
<sub>contenido</sub>
```

### ✔️ Ejemplo:

```html
<p>La fórmula química del agua es H<sub>2</sub>O.</p>
```

**Resultado visual esperado:**

> La fórmula química del agua es H₂O.

Esto es útil para fórmulas químicas, anotaciones científicas o variables indexadas.

---

## 🔹 Uso conjunto en expresiones

Estas etiquetas también se pueden usar juntas en expresiones más complejas. Por ejemplo:

### ✔️ Ejemplo:

```html
<p>La variable <var>x<sup>2</sup> + x<sub>1</sub></var> aparece en la ecuación.</p>
```

**Resultado visual esperado:**

> La variable x² + x₁ aparece en la ecuación.

---

## 🔸 Alternativas para notación matemática avanzada

Si bien `<sup>` y `<sub>` permiten representar contenido básico, cuando se trata de **fórmulas matemáticas complejas**, HTML por sí solo puede quedarse corto. En estos casos, se recomienda el uso de herramientas adicionales:

|Sistema|Descripción|
|---|---|
|**MathJax**|Biblioteca JavaScript que convierte expresiones matemáticas a SVG o HTML.|
|**MathML**|Lenguaje basado en XML para representar fórmulas matemáticas semánticamente.|
|**Latex.js**|Convierte código LaTeX a HTML5 para mostrar matemáticas en la web.|

Estos sistemas permiten una notación mucho más rica, semántica y visualmente fiel al estándar científico.

---

## 🔸 Accesibilidad y buenas prácticas

- **Evitar el uso visual forzado:** no es recomendable usar estilos CSS para simular superíndices o subíndices (`vertical-align`) si HTML ya proporciona estas etiquetas semánticas.
    
- **Utilizar correctamente `<sup>` y `<sub>` mejora la accesibilidad**, ya que los lectores de pantalla pueden interpretar mejor el contenido si está correctamente marcado.
