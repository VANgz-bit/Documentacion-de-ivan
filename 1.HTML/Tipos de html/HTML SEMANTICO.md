El **HTML semántico** es una de las mejores prácticas modernas al escribir HTML. Consiste en **usar etiquetas que describen claramente el significado del contenido** que encierran. En otras palabras, en lugar de usar etiquetas genéricas como `<div>` o `<span>` para todo, usás etiquetas como `<header>`, `<article>`, `<footer>`, etc., que le dan **contexto y estructura lógica** al documento.
###  ¿Por qué es importante el HTML semántico?

1. **Mejor accesibilidad**: Los lectores de pantalla entienden mejor la estructura.
    
2. **SEO (posicionamiento en buscadores)**: Google y otros motores comprenden mejor el contenido.
    
3. **Código más claro**: Más fácil de leer, mantener y colaborar.
    
4. **Futuro asegurado**: Cumple con los estándares de HTML5.
    

### Etiquetas semánticas comunes

| Etiqueta                  | Significado / Uso                                         |
| ------------------------- | --------------------------------------------------------- |
| `<header>`                | Encabezado principal de una página o sección              |
| `<nav>`                   | Contiene enlaces de navegación                            |
| `<main>`                  | Contenido principal de la página (solo uno por documento) |
| `<section>`               | Sección temática del documento                            |
| `<article>`               | Contenido independiente (una noticia, blog, post, etc.)   |
| `<aside>`                 | Contenido secundario, como una barra lateral              |
| `<footer>`                | Pie de página, puede ir al final de una página o sección  |
| `<figure>`                | Contenedor de una imagen o elemento multimedia            |
| `<figcaption>`            | Descripción de la figura                                  |
| `<mark>`                  | Resalta texto relevante                                   |
| `<time>`                  | Fecha u hora                                              |
| `<summary>` y `<details>` | Crear contenido expandible/colapsable                     |
