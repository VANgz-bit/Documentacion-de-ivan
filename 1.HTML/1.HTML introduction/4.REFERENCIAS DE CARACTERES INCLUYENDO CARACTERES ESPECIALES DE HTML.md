	En HTML, los caracteres `<`, `>`, `"`, `'`y `&`son caracteres especiales. Forman parte de la sintaxis HTML. ¿Cómo se incluye uno de estos caracteres especiales en el texto? Por ejemplo, si se desea usar un símbolo "&" o "menor que" sin que se interprete como código.

Esto se logra mediante [referencias de caracteres](https://developer.mozilla.org/en-US/docs/Glossary/Character_reference) . Se trata de códigos especiales que representan caracteres y que se utilizan en estas circunstancias específicas. Cada referencia de carácter comienza con un símbolo & y termina con un punto y coma (;).

| Carácter literal | Referencia de personaje equivalente |
| ---------------- | ----------------------------------- |
| <                | `&lt;`                              |
| >                | `&gt;`                              |
| "                | `&quot;`                            |
| '                | `&apos;`                            |
| &                | `&amp;`                             |

El equivalente de la referencia de carácter se puede recordar fácilmente porque el texto que utiliza puede interpretarse como menor que para [texto faltante] `&lt;`, entrecomillado para [texto faltante] `&quot;`y similar para otros. Para obtener más información sobre las referencias a entidades, consulte [la Lista de referencias a entidades de caracteres XML y HTML](https://en.wikipedia.org/wiki/List_of_XML_and_HTML_character_entity_references) (Wikipedia).

En el siguiente ejemplo, hay dos párrafos:

```
<p>In HTML, you define a paragraph using the <p> element.</p>

<p>In HTML, you define a paragraph using the &lt;p&gt; element.</p>
```

En la salida en vivo a continuación, se puede ver que el primer párrafo está mal. El navegador interpreta la segunda instancia `<p>`como el inicio de un nuevo párrafo. El segundo párrafo se ve bien porque tiene corchetes angulares con referencias a caracteres.
