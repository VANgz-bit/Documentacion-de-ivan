Los atributos `srcset` y `sizes` son parte de HTML y se utilizan principalmente con la etiqueta `<img>` para mejorar el **diseño responsive** de las imágenes. Permiten que el navegador seleccione automáticamente la mejor imagen según el tamaño de pantalla y la resolución del dispositivo. Vamos uno por uno:

## ¿QUE ES SRCSET?

El atributo ==srcset==  permite especificar múltiples versiones de una imagen para que el navegador elija la mas adecuada según el tamaño de pantalla o la densidad de pixeles del dispositivo.

EJEMPLO:

En html :

`<img` 
	`src="imagen-mediana.jpg"`
	`srcset="imagen-pequeña.jpg 480w, imagen-mediana.jpg 800w, imagen-grande.jpg 1200w">`


La imagen-pequeña.jpg 400w significa que esa imagen tiene 480 pixeles de ancho.

El navegador elige la mejor opción según el ancho de pantalla o contenedor.


## ¿QUE ES SIZES?


El atributo sizes le dice al navegador cuanto espacio ocupara la imagen en la pantalla
, dependiendo del ancho del viewport (pantalla)


EJEMPLO:

HTML:

`<img src="imagen-mediana.jpg" srcset="imagen-pequeña.jpg 480w, imagen-mediana.jpg 800w, imagen-grande.jpg 1200w" sizes="(max-width: 600px) 480px, (max-width:900px) 800px, (max-width: 900px) 800px, 1200px">`

 (max-width: 600px) 480px : si la pantalla tiene 600px o menos la imagen ocupara 480px de ancho.

 (max-width: 900px) 800px :si tiene hasta 900px, usara 800px.

 1200px es el valor por defecto, si no se cumple ninguna de las condiciones anteriores.
