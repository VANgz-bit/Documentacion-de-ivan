 Las media queries (consultas de medios )  son una parte fundamental de css que se usan para aplicar estilos condicionales según ciertas características del dispositivo, como el tamaño de pantalla, resolución, orientación, etc. 

## ¿PARA QUE SIRVE?


Sirven para crear diseños responsivos, es decir, paginas web que se adapten a diferentes tamaños de pantalla, celulares, tablets, laptops, monitores, etc.

SINTAXIS BASICA :

@media not | only mediatype and (expressions) {
	css-code
}

EXPLICACION

1. @media: palabra clave que inicia la media query.
	
2. not | only son operadores opcionales; Not (sirve para negar una condición, es decir, los estilos se aplicaran si no se cumple lo que digo); Only los estilos solo se agregaran si el dispositivo admite media queries.
	
3. mediatype (opcional) : indicamos el tipo de dispositivo (screen, print, all).
	
4. and: es el operador lógico para combinar condiciones.
	
5. media-feature o expressions: es la característica que elegimos (max-width-orientation,etc)
