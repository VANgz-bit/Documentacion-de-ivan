
## ¿QUE ES UNA FEATURE QUERIES?

Las features queries (consultas de características) que en css es una herramienta que nos permite aplicar estilos solo si el navegador soporta ciertas propiedades o valores en css. 

SINTAXIS BASICA:

@SUPPORTS (PROPIEDAD: VALOR ) {
    estilos que se aplican solo si el navegador soporta esta propiedad  y valor
}

EJEMPLO:

@SUPPORTS (DISPLAY: GRID ){
	.CONTENEDOR{
	display: grid
	}
}

si el navegador entiende display: grid, entonces aplica los estilos.

## ¿PARA QUE SIRVE?

1. Compatibilidad progresiva : usamos lo ultimo de css sin romper el diseño de navegadores viejos.
	
2. Mejor rendimiento y organización 
	
3. Evitar hacks de css o condicionales con JavaScript
