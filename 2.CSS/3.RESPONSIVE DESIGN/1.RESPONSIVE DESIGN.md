### ¿Qué es el Responsive Design?

Es una técnica de diseño web que permite que un sitio se **adapte automáticamente** a diferentes tamaños de pantalla: móviles, tablets, laptops, etc.  
El objetivo es que el sitio sea **fácil de usar y se vea bien** sin importar el dispositivo.

### ¿Cómo se logra?

#### 1. **Media Queries** (Consultas de medios)

Son reglas CSS que aplican estilos dependiendo del **ancho de pantalla** (o resolución del dispositivo).

#### 2. **Unidades relativas**

En lugar de usar `px`, se usan unidades como:

- `em` o `rem`: relativas al tamaño de fuente.
    
- `%`: relativo al contenedor padre.
    
- `vw` / `vh`: relativo al ancho y alto de la pantalla.

#### 3. **Flexbox y Grid**

Son sistemas de layout muy usados porque **se adaptan fácilmente** a diferentes tamaños de pantalla.
