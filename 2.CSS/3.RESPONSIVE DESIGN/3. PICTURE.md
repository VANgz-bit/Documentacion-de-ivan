
## ¿QUE ES `<PICTURE>`?

El elemento picture nos permite definir diferentes imágenes para diferentes condiciones, como el ancho de la pantalla, la orientación o incluso el tipo de imagen (.webp vs .jpg)

ESTRUCTURA BASICA:

HTML:

`<picture>`
	`<source srcset="imagen-grande.webp" type="image/webp" media="(max-width: 800px)">`
	`<source srcset="imagen-mediana.jpg" type="image/jpg" media="(max-width: 600px)">`
	`<img src="imagen-pequeña.jpg" alt="ejemplo">`
`</picture>`


`<source>`define una condición (el ancho de la pantalla) y que imagen usar si se cumple dicha condición.

==media==: funciona igual que en css. Se usa para aplicar reglas según el tamaño del viewport.

==type:== indica el formato de la imagen, por ejemplo image/webp, para que el navegador use esa versión si la soporta.

`<img>` es la imagen por defecto que se muestra si ninguna condición de <`source`> se cumple.
