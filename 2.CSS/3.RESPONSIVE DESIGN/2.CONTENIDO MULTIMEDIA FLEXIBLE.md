### IMAGENES FLEXIBLES

Hay muchas formas de hacer que una imagen sea flexible y se adapte a la pantalla dependiendo de que queramos.

## 1. **Imagen como contenido (`<img>`) con CSS responsivo**

Ideal si estás usando la imagen como **parte del contenido**, no como fondo.

img {
  width: 100%;
  height: auto; /* Mantiene la proporción */
  display: block;
}

## 2. **Imagen de fondo (`background-image`) en CSS**

Ideal si querés que la imagen **cubra toda la pantalla** como fondo decorativo.

body {
  margin: 0;
  height: 100vh;
  background-image: url('tu-imagen.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
}

Estas son las mas sencillas pero existen mas métodos que tienen otro apartado específico.


## VIDEOS FLEXIBLES

Para los videos también existen muchas formas de adaptarlos pero por ahora solo veremos una.

.content{

	max-width: 100%;

	 max-height: 575px;

	   aspect-ratio: 16/9;

	   margin: 20px;

}

video{

	   width: 100%;

	   max-width: 800px;

}

Con esta configuración:

- El contenedor `.content` puede estirarse hasta el 100% del ancho disponible, pero no lo excede.
- `aspect-ratio: 16 / 9` mantiene la proporción del video, sin importar el tamaño.
- `max-height: 575px` limita el alto máximo del video.
- El `<video>` ocupa el 100% del ancho de su contenedor, pero no supera los 800px gracias a `max-width`.

Esto nos permite que el video **sea responsivo y mantenga proporciones**, adaptándose a distintos tamaños de pantalla sin deformarse.
