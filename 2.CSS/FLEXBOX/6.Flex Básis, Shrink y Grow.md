
Estas tres propiedades determinan **cómo se comporta un elemento flexible en cuanto a tamaño**. Son el corazón del sistema de dimensionamiento en Flexbox.
Estas propiedades **solo funcionan en elementos hijos de un contenedor `display: flex`**

## `flex-basis`: el tamaño base del ítem

###  ¿Qué hace?

Define el **tamaño inicial** del ítem **antes** de aplicar `flex-grow` o `flex-shrink`.

- Si usas `flex-basis: auto`, tomará el ancho o alto definido por `width` o `height`.
    
- Puedes usar unidades como `px`, `%`, `em`, etc.

EJEMPLO

.item {
  flex-basis: 200px;
}

## `flex-grow`: cuánto puede **crecer** un ítem

### 📌 ¿Qué hace?

Controla cuánto puede **expandirse** un ítem para ocupar **espacio extra disponible** en el eje principal.

- El valor es un número **relativo**.
    
- Un ítem con `flex-grow: 2` crecerá el doble que uno con `flex-grow: 1`.
    

### Ejemplo:

.item {
  flex-grow: 1;
}


## `flex-shrink`: cuánto puede **encogerse** un ítem

### 📌 ¿Qué hace?

Controla cuánto puede **reducirse** un ítem si **no hay suficiente espacio** en el eje principal.

- También es un valor relativo.
    
- Un ítem con `flex-shrink: 2` se encoge el doble que uno con `flex-shrink: 1`.
    

### Ejemplo:

.item {
  flex-shrink: 1;
}

## Shorthand: `flex`

Puedes escribir estas tres propiedades juntas con la propiedad abreviada **`flex


Ejemplos:

.item {
  flex: `<grow> <shrink> <basis>;`
}



flex: 1 1 0;      /* crece y se encoge, tamaño base 0 */
flex: 0 1 auto;   /* solo se encoge, tamaño base por defecto */
flex: 1;          /* equivalente a: 1 1 0 (muy común) */
