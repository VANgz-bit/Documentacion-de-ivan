# Introducción a CSS Flexbox

**Flexbox** (Flexible Box Layout) es un sistema de diseño unidimensional en CSS que te permite **distribuir, alinear y organizar elementos** dentro de un contenedor de forma más eficiente y predecible que los modelos antiguos como `float` o `inline-block`.

---

##  TEMAS A ENCONTRAR

Aquí están los **contenidos principales** que debes dominar:

### 1.  Activar Flexbox

- `display: flex;` y `display: inline-flex;`
    
- Qué ocurre cuando activas Flexbox en un contenedor.
    

---

### 2.  Ejes de Flexbox: **Main Axis y Cross Axis**

- Qué son y cómo funcionan.
    
- Cómo cambian según `flex-direction`.
    
- Relación con `justify-content` (main axis) y `align-items` (cross axis).
    

---

### 3.  Dirección y flujo de los elementos

- `flex-direction` → dirección del main axis.
    
- `flex-wrap` → permite que los elementos se ajusten en varias líneas.
    
- `flex-flow` → abreviatura de las dos anteriores.
    

---

### 4.  Distribución de espacio en el eje principal

- `justify-content`
    
    - `flex-start`
        
    - `flex-end`
        
    - `center`
        
    - `space-between`
        
    - `space-around`
        
    - `space-evenly`
        

---

### 5.  Alineación en el eje cruzado

- `align-items` → para todos los elementos.
    
- `align-self` → para un solo elemento.
    
- `align-content` → para múltiples líneas (cuando hay `wrap`).
    

---

### 6.  Tamaño y crecimiento de los elementos

- `flex` (shorthand de `flex-grow`, `flex-shrink` y `flex-basis`)
    
- `flex-grow` → cuánto crece un elemento.
    
- `flex-shrink` → cuánto se reduce.
    
- `flex-basis` → tamaño base inicial.
    

---

### 7.  Orden de los elementos

- `order` → cambia el orden visual sin alterar el HTML.
    

---

### 8. 🎨 Espaciado entre elementos

- `gap` → espacio entre elementos flexibles sin usar márgenes.
