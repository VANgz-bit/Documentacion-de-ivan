## `flex-direction`

Define **la dirección principal** en la que se colocarán los elementos dentro del contenedor.

### Valores posibles:

|Valor|Descripción|
|---|---|
|`row`|Los elementos se colocan en una **fila horizontal** (de izquierda a derecha). _(Por defecto)_|
|`row-reverse`|Igual que `row`, pero en orden **reverso** (de derecha a izquierda).|
|`column`|Los elementos se colocan en una **columna vertical** (de arriba a abajo).|
|`column-reverse`|Igual que `column`, pero en orden **reverso** (de abajo a arriba).|

### Ejemplo:

`.contenedor {
display: flex;
flex-direction: row; /* o column, row-reverse, etc. */
}`

---

## 📦 `flex-wrap`

Controla si los elementos deben mantenerse en una sola línea o **envolverse** si no hay espacio.

### Valores posibles:

|Valor|Descripción|
|---|---|
|`nowrap`|Todos los elementos en **una sola línea**. _(Por defecto)_|
|`wrap`|Los elementos se **ajustan a múltiples líneas**, de arriba hacia abajo.|
|`wrap-reverse`|Igual que `wrap`, pero las líneas se colocan **de abajo hacia arriba**.|

### Ejemplo:

`.contenedor {
display: flex;
flex-wrap: wrap;
}`

---

## 🔗 `flex-flow`

Es una **abreviación** que combina `flex-direction` y `flex-wrap`.


`.contenedor {
flex-flow: 
`<flex-direction> <flex-wrap>`;
}`

## En resumen:

|Propiedad|¿Qué hace?|
|---|---|
|`flex-direction`|Controla la **dirección principal** de los elementos.|
|`flex-wrap`|Decide si los elementos **se ajustan a múltiples líneas**.|
|`flex-flow`|Combina ambas propiedades para simplificar el código.|
