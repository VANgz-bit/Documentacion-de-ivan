La propiedad **`order`** te permite **cambiar el orden visual** de los elementos flexibles **sin modificar el HTML**.

Por defecto, los elementos flex hijos se muestran **en el orden en que están en el HTML**, de izquierda a derecha (o de arriba a abajo si usas `flex-direction: column`).

Con `order`, puedes **reordenarlos visualmente**, y Flexbox se encargará del resto.

Todos los elementos tienen `order: 0` por defecto. Al cambiar este valor, puedes mover un ítem hacia adelante o atrás en el layout.

## Importante

- `order` **no cambia el orden en el DOM**, solo el **orden visual**.
    
- Es útil para **reordenar layouts responsivos** o **enfocar atención en elementos específicos** según el contexto.
    
- Puedes usar cualquier número positivo, negativo o cero: `order: -1`, `order: 10`, etc.


## Cuándo usar `order`

- En menús o grids que se reorganizan según el tamaño de pantalla.
    
- Para invertir el orden visual sin duplicar código HTML.
    
- En sitios accesibles, puedes mostrar un contenido primero visualmente pero dejarlo en otro lugar en el DOM por razones de accesibilidad.
