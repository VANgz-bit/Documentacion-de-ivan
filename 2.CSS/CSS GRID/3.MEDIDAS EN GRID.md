Cada elemento con el que se trabaja en css usa unidades de medidas para definir sus características a lo largo y ancho de su contenedor, en grid se suelen usar las mismas unidades de medidas exceptuando una en particular el   **==fr==**   (fraction/fracción)

### FR (fraction/fracción)

Esta forma de medir hace que el espacio disponible del contenedor se divida en fracciones proporcionales

EJEMPLO:
width: 300px;

	grid-template-: 1fr 2fr 2fr;

Se reparte el espacio total: 1 parte para la primer columna y 2 partes para la segunda y la tercera columna.

Dado que la caja contenedora-grid mide 300px, el navegador calcula cuanta porciones debe llevar cada columna, la primera llevara 60px la segunda y la tercera 120px entonces nos da un total de 300px porque,  60px + 120px +120px = 300px.

### AUTO


Auto adapta la columna o fila al contenido mas grande dentro de ella.

EJEMPLO:

	 grid-template-columns: auto auto;

las columnas se adaptaran en base a lo que su contenido tenga dentro de ella.

### MIN-CONTENT, MAX-CONTENT, MINMAX

#### MIN-CONTENT

Con esta función el tamaño se ajusta al mínimo ancho posible sin romper el contenido, en otras palabras lo hace tan angosto como pueda sin cortar el contenido.

EJEMPLO:

	grid-template-columns: min-content 1fr ;

como resultado la columna  se encoge al mínimo necesario para acomodar el texto saltando delinea si es necesario.


#### MAX-CONTENT

Acá el tamaño se ajusta al ancho máximo que necesita el contenido sin cortarlo, en otras palabras nos da tanto espacio como necesitemos para que el contenido se muestre en una linea


EJEMPLO:

	grid-template-columns: max-content 1fr;

Esto nos da como resultado que la columna se agranda al ancho completo del texto


### MINMAX (min-content/max-content)

Esta es una función que le dice al navegador que deber tener un valor mínimo de ancho y una valor máximo , es decir que tendrá una proporción mínima y una limite máximo de cuanto crecerá la proporción.

SIRVE PARA :

Evitar que una columna se vuelva demasiada pequeña (min) y permitir que crezca gradualmente hasta llegar a un máximo establecido (max)

SINTAXIS BASICA:

	GRID-CONTENT-COLUMN: MINMAX(100PX, 1FR);

entonces aclaramos que el tamaño mínimo que tendrá es de 100px y lo máximo que se agrandara será del tamaño disponible.


## REPEAT

Esta función como tal no se usa para medir sino para simplificar de manera eficiente todos los valores definidos, si por ejemplo tenemos con tenedor grid al cual queremos agregar muchas celdas tendríamos que escribir el mismo código repetidamente entonces usamos repeat, para definir cuantas veces se va a escribir el mismo valor.

SINTAXIS BASICA:

grid-template-columns: repeat (n, tamaño)
grid-template-row: repeat (n, tamaño)

"n" es el nro. de veces que se pueden repetir
"tamaño" es la medida que tendrá puede ser(px, %, fr, etc. )

EJEMPLO

	grid-template-columns: repeat (4, 300px)

Esto crea 4 columnas de 300px cada una.

También se pueden combinar valores:

	grid-template-column: repeat(3, 1fr) 2fr;

De esta forma definimos que vamos a tener 3 columnas que ocupan el tamaño disponible (1fr) y otra aparte que tendrá 2fr.

También podemos repetir patrones:

	grid-template-column: repeat(2, 1fr 2f) ;

De esta forma declaramos que hay 4 columnas en el cual  se repiten 2 veces el patrón (1fr 2fr)


También se puede combinar con minmax;

	grid-template-column: repeat(minmax( 100px,300px)) ;

Entonces tendremos dos columnas que tendrán lo mínimo que se achicara será 100px y lo máximo que se agrandara será 300px.
