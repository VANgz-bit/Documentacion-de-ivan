En grid podemos posicionar como la grilla completa (grid-contenedor) como sus hijos (grid-items) que estos están dentro del algún contenedor a su vez.
En otras palabras podemos decir que;
Hay dos niveles en los que podemos alinear/distribuir elementos en CSS Grid:
1. **Alineación del contenido del grid (contenedor)**
2. **Alineación de los ítems dentro del grid (ítems)**

### Alineación del contenido (contenedor grid)

Estos afectan **todo el contenido dentro del grid**. Se usan en el contenedor `.container`.

|Propiedad|Eje|Qué alinea|
|---|---|---|
|`justify-content`|Horizontal (x)|Alinea **el contenido del grid** en X|
|`align-content`|Vertical (y)|Alinea **el contenido del grid** en Y|
Los valores que usamos para alinear o distribuir lo solemos usar en Flexbox, estos valores comunes son:

- `start`: lo posiciona al principio.
    
- `end`: lo posiciona al final.
    
- `center`: lo deja centrado.
    
- `space-between`: ajusta un espacio entre ítems.
	
- `space-around`: ajusta un espacio alrededor.
    
- `space-evenly`: deja un espacio igualado.

#### Ejemplo:

.container {
  display: grid;
  grid-template-columns: repeat(3, 100px);
  grid-template-rows: repeat(2, 100px);
  justify-content: center; /* centra horizontalmente el grid */
  align-content: center;   /* centra verticalmente el grid */
}

De esta manera dejamos el contenido grid centrado horizontal y verticalmente:

### Alineación de ítems dentro del grid

Estas propiedades afectan a **cada ítem dentro de sus celdas**. Se pueden usar tanto en el contenedor (para todos los ítems) como en ítems individuales.

| Propiedad       | Eje            | Qué alinea                                     |
| --------------- | -------------- | ---------------------------------------------- |
| `justify-items` | Horizontal (x) | Alinea **los ítems** dentro de sus celdas en X |
| `align-items`   | Vertical (y)   | Alinea **los ítems** dentro de sus celdas en Y |

#### Ejemplo

.container {
  display: grid;
  grid-template-columns: repeat(3, 100px);
  grid-template-rows: repeat(2, 100px);
  justify-items: center; /* ítems centrados horizontalmente */
  align-ítems: center;   /* ítems centrados verticalmente */
}
Esto centra a cada ítem dentro de sus celdas.

### Alineación por ítem

Cuando queremos alinear un solo ítem dentro de la grilla usamos otras propiedades llamadas:

| Propiedad      | Eje            | Qué alinea                                     |
| -------------- | -------------- | ---------------------------------------------- |
| `justify-self` | Horizontal (x) | Alinea **los ítems** dentro de sus celdas en X |
| `align-self`   | Vertical (y)   | Alinea **los ítems** dentro de sus celdas en Y |
.item__grid:last-child {
  justify-self: end;
  align-self: start;
}
De esta forma lo ubicamos solo al primer hijo elemento, dentro de su celda arriba al final en la derecha.

## Diferencias

|Propiedad|Aplica a...|Dirección|Qué hace|
|---|---|---|---|
|`justify-content`|Contenedor|Horizontal|Distribuye el **grid como conjunto**|
|`align-content`|Contenedor|Vertical|Distribuye el **grid como conjunto**|
|`justify-items`|Contenedor/ítem|Horizontal|Alinea los ítems **dentro de su celda**|
|`align-items`|Contenedor/ítem|Vertical|Alinea los ítems **dentro de su celda**|
|`justify-self`|Ítem individual|Horizontal|Alinea ese ítem específico en su celda|
|`align-self`|Ítem individual|Vertical|Alinea ese ítem específico en su celda|
