
## ¿Qué es CSS Grid?

**CSS Grid** es un sistema de diseño en CSS que permite crear **distribuciones de elementos en dos dimensiones**, es decir, en filas y columnas. A diferencia de Flexbox, que está orientado principalmente a una sola dimensión (horizontal o vertical), Grid te da un control mucho más completo sobre la disposición de elementos en el eje **horizontal y vertical al mismo tiempo**.

### ¿Por qué usar Grid?

Grid es ideal cuando necesitas:

- Diseñar **layouts complejos** como rejillas, tableros, páginas completas o secciones con múltiples columnas y filas.
    
- Tener **alineación precisa** de elementos tanto en columnas como en filas.
    
- Controlar la **posición de elementos** de manera intuitiva con pocas líneas de código.


## Ejemplo Básico

html

`<div class="grid-container">`
	  `<div> 1</div>`
	  `<div> 2</div>`
	  `<div> 3</div>`
	  `<div> 4</div>`
`</div>`

EN CSS

.grid-container {
  display: grid;
  grid-template-columns: 200px 200px; /* Dos columnas de igual tamaño */
  gap: 10px; /* Espacio entre celdas */
}


## PARTES DEL GRID

### GRID CONTAINER

El grid container (ejemplo: div, ul, main; etc.) es el elemento padre que se define como un contenedor de cuadrilla, es decir, el elemento que le definamos display: grid. pasara a ser grid container.
![[Pasted image 20250527190939.png]]

### GRID-ITEMS

Son los elementos que se encuentran dentro de la cuadrilla y son los hijos directos del elemento contendor, cada uno de estos elementos puede ocupar uno o mas celdas de la cuadricula

![[Pasted image 20250527191415.png]]


### LINEAS DE LA CUADRICULA

Son los bordes las cuadriculas y separan los bordes de las columnas y las filas.

![[Pasted image 20250527191817.png]]


### GRID TRACKS

Son espacios entres dos líneas del grid; Una es grid row (una pista horizontal) y  otra es grid column (una pista vertical)


### CELDAS DE LA CUADRICULAS

Las celdas es el espacio real de dicho cuadrante en la que se encuentra, es decir, es la unidad base donde se pueden colocar los elementos, no confundir con grid-ítem, grid-item es el elemento html que se encuentra dentro del la celda.

![[Pasted image 20250527192551.png]]

### GRID AREA

Los grids área  son un conjunto de una o mas celdas y se utiliza para ubicar elementos de maneras mas eficientes dentro de las cuadricula
