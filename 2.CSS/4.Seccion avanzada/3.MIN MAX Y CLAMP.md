
## ¿Qué son `min()`, `max()` y `clamp()`?

Son **funciones matemáticas de CSS** que permiten establecer **valores dinámicos y controlados** para propiedades como `width`, `height`, `font-size`, `padding`, `margin`, etc.

Estas funciones hacen que el diseño sea más **responsive, flexible y adaptable**, combinando unidades como `px`, `%`, `vw`, `em`, etc.

---

## 1. **`min()`**

Devuelve el **valor más pequeño** de entre varios que le pases.

### Sintaxis:

min(valor1, valor2, valor3, ...)

### Ejemplo:

`width: min(100%, 500px);`

**¿Qué hace?**  
La anchura será **el menor valor** entre 100% del contenedor o 500px.  
Sirve para **evitar que algo crezca demasiado**.


## 2. **`max()`**

Devuelve el **valor más grande** de entre varios que le pases.

### Sintaxis:

`max(valor1, valor2, valor3, ...)`

### Ejemplo:

`font-size: max(16px, 2vw);`

**¿Qué hace?**  
El tamaño del texto será **al menos 16px**, pero **crecerá** si `2vw` es mayor.  
Se usa para asegurar **mínimos legibles** en tamaños responsive.

## 3. **`clamp()`**

Esta es la más poderosa. Establece un valor **que puede crecer o achicarse dentro de un rango definido**. Mezcla lo mejor de `min()` y `max()` en una sola función.

### Sintaxis:

`clamp(valor-mínimo, valor-flexible, valor-máximo)`

- `valor-mínimo`: el valor más pequeño que puede tener.
    
- `valor-flexible`: un valor que puede cambiar (normalmente con unidades como `vw`).
    
- `valor-máximo`: el tope que no puede superar.
    

### Ejemplo:

`font-size: clamp(14px, 2vw, 24px);`

**¿Qué hace?**

- Nunca será menor a 14px.
    
- Escala con el ancho de la ventana (`2vw`).
    
- Pero nunca será mayor a 24px.  
    Sirve mucho para **tamaños de texto, márgenes, paddings o anchos responsivos sin media queries**.


## Comparación rápida

| Función   | ¿Qué hace?                                     | ¿Cuándo usarla?                                           |
| --------- | ---------------------------------------------- | --------------------------------------------------------- |
| `min()`   | Toma el valor más chico                        | Para **limitar crecimiento**                              |
| `max()`   | Toma el valor más grande                       | Para **asegurar mínimos**                                 |
| `clamp()` | Toma un valor dentro de un **rango adaptable** | Para valores que deben escalar pero **dentro de límites** |


## Ejemplos reales:

**1. Ancho máximo de un contenedor**

`.container {   width: min(100%, 1200px); }`

**2. Texto que nunca sea muy chico ni muy grande**


`h1 {   font-size: clamp(1.5rem, 5vw, 3rem); }`

**3. Margen adaptable**

`main {   margin: max(1rem, 5%); }`
