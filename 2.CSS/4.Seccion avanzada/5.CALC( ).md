## 1. ¿Qué es `calc()`?

`calc()` es una **función nativa de CSS** que permite realizar **cálculos matemáticos directamente dentro de las propiedades de estilo**. Es útil cuando necesitás combinar diferentes unidades (como `px` con `%` o `em`) o cuando querés definir valores **flexibles y precisos** que no se pueden lograr con una sola unidad.

---

## 2. ¿Para qué se usa?

Se usa para establecer **valores calculados dinámicamente** en propiedades como:

- `width`, `height`
    
- `margin`, `padding`
    
- `top`, `left`, `right`, `bottom`
    
- `font-size`
    
- y muchas más.
    

Esto te da más control al crear **layouts responsive o adaptativos**, o cuando necesitás ajustar medidas con precisión.

---

## 3. Sintaxis

`propiedad: calc(expresión);`

Donde **expresión** es una operación matemática que puede mezclar valores numéricos y unidades de CSS.

---

## 4. Operaciones soportadas

`calc()` permite las siguientes operaciones básicas:

|Operación|Símbolo|Ejemplo|
|---|---|---|
|Suma|`+`|`width: calc(100% + 20px)`|
|Resta|`-`|`height: calc(100vh - 60px)`|
|Multiplicación|`*`|`width: calc(2 * 1rem)`|
|División|`/`|`font-size: calc(16px / 2)`|

**Importante**:

- Siempre debe haber **espacios entre números, operadores y unidades**.
/* Correcto */
width: calc(100% - 50px);

/* Incorrecto */
width: calc(100%-50px);


## 5. Compatibilidad de unidades

Podés mezclar diferentes **unidades compatibles** en la misma operación:

`width: calc(50% - 2rem);`

Esto es muy útil para diseños donde una parte es fija (como `px` o `rem`) y otra es relativa (`%` o `vh`).


## 6. Usos comunes

### a. Layouts flexibles con márgenes fijos


.container {
  width: calc(100% - 40px);
  margin: 0 20px;
}

### b. Centrar elementos exactamente


.left-box {
  position: absolute;
  left: calc(50% - 100px); /* Centra un elemento de 200px */
}

### c. Ajustar altura menos una cabecera


.main {
  height: calc(100vh - 60px); /* 60px es el header */
}


### d. Junto con `var()`

Podemos usar `calc()` junto a variables CSS:

:root {
  --gutter: 1rem;
}

.card {
  padding: calc(var(--gutter) * 2);
}


## 7. Cosas a tener en cuenta

- `calc()` **no se puede anidar**: no podés hacer `calc(calc(...))`.
    
- Es evaluado **por el navegador en tiempo de renderizado**, no es una precomputación como en Sass.
    
- No se puede usar dentro de media queries directamente (aunque sí como parte del valor):


/* Incorrecto */
@media (min-width: calc(100% - 50px)) {}

/* Correcto */
.box {
  width: calc(100% - 50px);
}


## Conclusión

La función `calc()` es una herramienta extremadamente útil en CSS moderno, especialmente cuando trabajamos con **layouts fluidos o responsive**. Permite mezclar unidades y aplicar lógica matemática sin necesidad de JavaScript o preprocesadores. Cuando se combina con variables CSS (`var()`), se vuelve aún más poderosa para crear diseños adaptativos y limpios.
