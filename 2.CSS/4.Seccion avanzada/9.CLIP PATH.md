
## ¿Qué es `clip-path`?

`clip-path` es una propiedad de CSS que permite **definir una región de recorte** para un elemento. Todo lo que quede **fuera de esa forma definida se oculta visualmente**, como si estuviera recortado.  
El contenido sigue existiendo en el DOM, pero **no es visible ni clickeable fuera de la forma**.

---

## ¿Cómo funciona?

La propiedad `clip-path` define una **forma vectorial** (como un círculo, polígono, etc.) que se usa como **máscara de recorte** sobre un elemento. La región visible será la que esté **dentro** de esa forma.

```css
elemento {
  clip-path: circle(50%);
}
```

---

## Formas básicas compatibles con `clip-path`

CSS permite recortar con varias **formas geométricas predefinidas**:

### 1. `inset()`

Recorta un **rectángulo** desde los bordes (como un padding inverso).

```css
clip-path: inset(10px 20px 30px 40px);
```

- Orden: `top right bottom left`
    
- Podés agregar un **radio** para bordes redondeados:
    

```css
clip-path: inset(10px round 20px);
```

### 2. `circle()`

Recorta en forma de **círculo**.

```css
clip-path: circle(50% at 50% 50%);
```

- `50%`: radio del círculo.
    
- `at 50% 50%`: posición del centro (X Y), donde `50% 50%` es el centro del elemento.
    

### 3. `ellipse()`

Recorta en forma de **elipse** (óvalo).

```css
clip-path: ellipse(50% 30% at 50% 50%);
```

- `50% 30%`: radios X e Y.
    
- `at`: posición del centro.
    

### 4. `polygon()`

Permite recortar con **formas personalizadas** usando puntos.  
Los puntos se definen como coordenadas X e Y relativas al tamaño del elemento.

```css
clip-path: polygon(0% 0%, 100% 0%, 100% 100%, 0% 100%);
```

- Este ejemplo es un rectángulo.
    
- Podés crear triángulos, estrellas, trapecios, etc.
    

---

## También se puede usar con SVG

Además de las funciones anteriores, se puede usar una **ruta SVG (Scalable Vector Graphics)** como forma de recorte.

```css
clip-path: url(#miClip);
```

En este caso, la forma de recorte se define en una etiqueta `<clipPath>` dentro de un archivo o bloque SVG.

---

## Ejemplo con SVG:

```html
<svg width="0" height="0">
  <clipPath id="miClip">
    <circle cx="50" cy="50" r="40" />
  </clipPath>
</svg>

<div style="clip-path: url(#miClip);">
  <!-- contenido recortado en círculo -->
</div>
```

---

## ¿Qué se puede recortar?

- **Imágenes**
    
- **Cajas de texto o divs**
    
- **Elementos HTML en general**
    
- También se puede usar junto con `background`, `hover`, `transition`, etc.
    

---

## Relación con otras propiedades

### 1. `mask` vs `clip-path`

|Propiedad|Función principal|Soporta transparencias|
|---|---|---|
|`clip-path`|Recorta el área visible con formas|NO|
|`mask`|Oculta o muestra según niveles alfa|SÍ (más detallado)|

- `clip-path` es **todo o nada**: lo que está fuera se esconde por completo.
    
- `mask` permite **niveles de opacidad**, ideal para efectos más suaves o graduales.
    

### 2. `transform` y `clip-path`

Ambas pueden usarse juntas. Ejemplo: animar una forma recortada mientras se mueve.

```css
div {
  clip-path: circle(0% at 50% 50%);
  transition: clip-path 0.5s ease;
}

div:hover {
  clip-path: circle(75%);
}
```

---

## Compatibilidad

- Muy bien soportado en **navegadores modernos**: Chrome, Firefox, Edge, Safari.
    
- Algunas formas como `polygon()` o `circle()` funcionan en la mayoría de los dispositivos.
    
- Para soporte en versiones antiguas, puede ser necesario un **fallback** o usar SVG como alternativa.
    

---

## Buenas prácticas

- Siempre asegurarse de que el contenido dentro de `clip-path` no afecte negativamente la accesibilidad (ya que los elementos recortados no son visibles ni interactuables).
    
- Usar `clip-path` con `hover`, `focus`, `scroll`, etc., para crear **efectos visuales** atractivos (como animaciones o transiciones).
    
- Para animaciones más complejas, se puede combinar con **JS o SVG animado**.
    

---

## Comparativa visual (imaginaria):

**Imaginemos que tenemos un `div` de 200x200px con una imagen**.

- Con `clip-path: circle(50% at 50% 50%)`, solo verás un círculo central.
    
- Con `clip-path: inset(20px)`, verás un rectángulo más chico centrado.
    
- Con `clip-path: polygon(...)`, podés ver una estrella, un triángulo, etc.
    
- Con `mask`, podrías ver un degradado que deja pasar solo parte de la imagen suavemente.
    

---

## Resumen final

- `clip-path` = "recorte visual" de un elemento.
    
- Usa formas geométricas (`circle`, `ellipse`, `polygon`, `inset`) o formas SVG.
    
- Oculta completamente lo que queda fuera de la forma.
    
- No afecta el DOM, pero **elimina interactividad en lo que se recorta**.
    
- Excelente para diseños creativos, efectos visuales y animaciones.
