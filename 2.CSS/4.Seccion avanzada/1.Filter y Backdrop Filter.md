## **1. ¿Qué es `filter` en CSS?**

`filter` es una propiedad que **aplica efectos visuales (como desenfoque, brillo o escala de grises)** directamente a un **elemento HTML** (como una imagen, un div, etc.). Es similar a los filtros que ves en programas como Photoshop, pero aplicado en el navegador.

### **Sintaxis básica:**
selector {
  filter: función(valor);
}

### **Funciones que acepta `filter`:**

Las funciones de `filter` se pueden aplicar solas o combinadas (separadas por espacios):

#### 1. `blur(px)`

- Aplica un **desenfoque**.
    
- Cuanto mayor el valor en píxeles, más borroso se ve.
Ejemlpo:

filter: blur(5px);

#### 2. `brightness(%)` o `brightness(valor decimal)`

- Ajusta el **brillo**.
    
- `1` es el valor original.
    
- `>1` hace que el elemento sea más brillante.
    
- `<1` lo oscurece.


`filter: brightness(1.2); /* 120% */`

#### 3. `contrast(%)` o `contrast(valor)`

- Cambia el **contraste**.
    
- `1` es normal. `>1` lo incrementa, `<1` lo reduce.


`filter: contrast(0.8); /* Reduce contraste */`

#### 4. `grayscale(%)` o `grayscale(valor)`

- Convierte a **escala de grises**.
    
- `1` o `100%` es completamente en gris, `0` es a color.

`filter: grayscale(1);`

#### 5. `sepia(%)`

- Da un tono marrón antiguo, como una **foto antigua**.

`filter: sepia(1);`

#### 6. `invert(%)`

- Invierte los colores (blanco ↔ negro, etc.).

`filter: invert(100%);`

#### 7. `saturate(%)`

- Controla la **intensidad de los colores**.
    
- `1` es normal, `0` es gris.
    
- `>1` hace los colores más fuertes.

`filter: saturate(2);`

#### 8. `hue-rotate(grados)`

- Rota los colores en la rueda de color.

`filter: hue-rotate(180deg);`

#### 9. `drop-shadow(offset-x offset-y blur color)`

- Aplica una sombra **similar a box-shadow** pero en el contenido renderizado.

`filter: drop-shadow(5px 5px 10px black);`

---

### **Combinar filtros:**

Puedes aplicar varios filtros a la vez:


`filter: brightness(1.2) blur(3px) grayscale(0.5);`


## **2. ¿Qué es `backdrop-filter`?**

`backdrop-filter` es similar a `filter`, pero en vez de aplicarse al **elemento mismo**, se aplica al **fondo detrás del elemento**.

### **Uso típico:**

- Ideal para crear **efectos de vidrio (glassmorphism)**.
    
- Requiere que el elemento tenga **fondo semi-transparente** y **un contexto de fondo detrás**.
    

### **Ejemplo básico:**

.glass {
  background-color: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
}

Este ejemplo crea un efecto "vidrio esmerilado" sobre lo que está detrás.

---

### **Funciones de `backdrop-filter`**

Usa las **mismas funciones** que `filter`, pero aplicadas **al fondo detrás del elemento**, no al contenido del elemento.

Por ejemplo:

`backdrop-filter: brightness(0.5) blur(8px);`

---

### **Requisitos especiales:**

Para que `backdrop-filter` funcione:

1. El navegador debe soportarlo (es compatible con la mayoría moderna).
    
2. El elemento debe tener un **fondo semi-transparente** (`rgba`, `hsla` o `transparent`).
    
3. A veces se necesita `will-change: backdrop-filter;` o `contain: paint;` para optimizar.
    

---

## **Diferencias clave entre `filter` y `backdrop-filter`**

|Característica|`filter`|`backdrop-filter`|
|---|---|---|
|A qué se aplica|Al **elemento en sí mismo**|Al **contenido que está detrás**|
|Visibilidad del fondo|No afecta|Necesita fondo transparente|
|Soporte|Muy amplio|Menos consistente, pero en aumento|
|Usos comunes|Imágenes, botones, divs decorativos|Efecto vidrio, modales, menús flotantes|

---

## **Ejemplo completo con ambos filtros**

`<div class="contenedor">`
  `<div class="filtro">`
    `Texto con `filter``
  `</div>`
  `<div class="glass">`
    `Fondo con `backdrop-filter``
  `</div>`
`</div>`


### css

.contenedor {
  background-image: url('fondo.jpg');
  background-size: cover;
  height: 100vh;
  display: flex;
  gap: 20px;
  padding: 20px;
}

.filtro {
  filter: sepia(1) brightness(1.5);
  padding: 20px;
  background-color: white;
}

.glass {
  background-color: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px) contrast(1.2);
  padding: 20px;
}
