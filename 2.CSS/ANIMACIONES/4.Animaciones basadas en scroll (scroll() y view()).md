
Estas animaciones **no dependen del tiempo** como una animación normal (`2s`, `3s`, etc.), sino que **dependen del desplazamiento del scroll** o del momento en que un elemento **entra al viewport (pantalla)**.

### A. `scroll()`

- La animación se **sincroniza con el scroll del documento** (cuando hacés scroll vertical u horizontal).
    
- El elemento **avanza en su animación** a medida que scrolleás.

Ejemplo:

.elemento {
  animation-name: mover;
  animation-timeline: scroll();
}

**Sincroniza con el scroll del `body`** o del contenedor con overflow. Muy útil para efectos que se activan al hacer scroll.

### B. `view()`

- La animación se activa **cuando el elemento entra o sale del viewport (pantalla visible)**.
    
- Se comporta como un `IntersectionObserver`, pero **hecho solo con CSS**.

Ejemplo:

.elemento {
  animation-name: mover;
  animation-timeline: view();
}

**No depende del scroll global**, sino de **la visibilidad del elemento**.


## 2. **¿Qué es `animation-timeline`?**

Es una propiedad **nueva** en CSS que nos permite decirle a una animación **con qué debe sincronizarse**:

- Antes solo podíamos usar `animation-duration` y `animation-delay`, o sea, todo se movía con el tiempo.
    
- Con `animation-timeline`, ahora podemos usar **el scroll**, la visibilidad, o incluso una línea de tiempo personalizada para controlar la animación.

.elemento {
  animation-name: mover;
  animation-timeline: scroll(); /* o view() o una línea personalizada */
}


Si no usamos `animation-timeline`, la animación usa la línea de tiempo **por defecto (auto)**, es decir, el tiempo (`2s`, `5s`, etc.).

## 3. **¿Qué es `@scroll-timeline` y por qué lleva llaves?**

### ¿Qué es?

Es una **regla de CSS**, como `@keyframes`, que sirve para **definir una línea de tiempo personalizada** basada en el scroll.  
Se usa cuando necesitas más control que lo que da `scroll()` o `view()`.

Por eso **lleva llaves** `{}`: porque estás **definiendo propiedades** para esa línea de tiempo.

### Ejemplo:

@scroll-timeline miScroll {
  source: auto;               /* El scroll del documento */
  orientation: block;         /* Scroll vertical */
  scroll-offsets: start 0%, end 100%;
}
Acá **sí se usan llaves**, porque estamos **creando una línea de tiempo nueva** con nombre propio (`miScroll`).

## Entonces...

### `@scroll-timeline`

es una **herramienta de CSS** dentro del sistema de Scroll-Linked Animation.

## ¿Qué es una Scroll-Linked Animation?

Una **Scroll-Linked Animation** (animación vinculada al scroll) es cualquier animación en CSS o JavaScript que **no avanza con el tiempo**, sino que **avanza al ritmo del scroll**.

---

Existe una forma de usar el scroll, si no queremos que la animacion no se ligue al body el ponemos la animacion al contendor de donde provine y usamos prácticamente lo mismo pero con cambios notables.


## ¿Qué es `scroll-timeline-name`?

Es una **propiedad CSS nueva** que le **asigna un nombre de línea de tiempo de scroll a un elemento**.  
Es parecida a `scroll-timeline`, pero más directa y más compatible con la nueva sintaxis.

### Ejemplo:

.scroll-container {
  scroll-timeline-name: miScroll;
}

Esto **no crea la línea de tiempo**, solo le **da un nombre al scroll del elemento**, para que después lo podamos usar en `animation-timeline`.

## ¿Qué es `scroll-timeline-axis`?

Es una propiedad que **define la dirección** en la que se basa el scroll para animar.

|Valor|Dirección|
|---|---|
|`block`|Vertical|
|`inline`|Horizontal|
Ejemplo:

.scroll-container {
  scroll-timeline-name: miScroll;
  scroll-timeline-axis: block; /* vertical */
}


## ¿Cómo se conecta todo esto?

Con estas propiedades, **ya no necesitamos usar `@scroll-timeline` con llaves** para cosas simples.  
En vez de eso, podemos definir la línea de tiempo **directamente en el contenedor** con:

.scroll-container {
  scroll-timeline-name: miScroll;
  scroll-timeline-axis: block;
}

Y luego usarla así en el elemento que se va a animar:

.elemento-animado {
  animation-name: mover;
  animation-timeline: miScroll;
}


## Entonces:

| Propiedad              | ¿Qué hace?                                   | ¿Reemplaza a...?                                 |
| ---------------------- | -------------------------------------------- | ------------------------------------------------ |
| `scroll-timeline-name` | Le pone un nombre al scroll del contenedor   | Parte de `@scroll-timeline { source: ... }`      |
| `scroll-timeline-axis` | Define si el scroll es vertical u horizontal | Parte de `@scroll-timeline { orientation: ... }` |
