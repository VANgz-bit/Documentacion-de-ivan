
Las **transiciones** en CSS permiten que los cambios de propiedades CSS ocurran de forma **suave y gradual**, en lugar de forma instantánea. Es decir, cuando un valor de propiedad cambia (por ejemplo, el color de fondo o el tamaño), CSS puede animar ese cambio con una transición fluida.

Esto es especialmente útil para crear interfaces más elegantes, intuitivas y agradables para el usuario.

### Propiedades:

1. **`transition-property`**: La propiedad que será animada (`color`, `background-color`, `transform`, etc.).
    
2. **`transition-duration`**: Tiempo que dura la transición (por ejemplo, `0.3s` o `500ms`).
    
3. **`transition-timing-function`**: Controla la velocidad de la transición a lo largo del tiempo (por ejemplo, `ease`, `linear`, `ease-in`, `ease-out`, `cubic-bezier()`).
    
4. **`transition-delay`** _(opcional)_: Tiempo que se espera antes de comenzar la transición.


EJEMPLO

.button {
  background-color: blue;
  transition: background-color 0.4s ease-in-out;
}

.button:hover {
  background-color: red;
}
Cuando el usuario pasa el cursor por encima de .button, el color de fondo cambia gradualmente de azul a rojo en 0.4 segundos.


## Se pueden aplicar múltiples transiciones separadas por comas:

.box {
  transition: background-color 0.5s ease, transform 0.3s ease-in;
}

## Funciones de tiempo (`timing functions`)

La función de tiempo define la **aceleración** del efecto. Algunos ejemplos:

- `linear`: la transición ocurre a velocidad constante.
    
- `ease`: inicio lento, rápido en el medio, y final lento (valor por defecto).
    
- `ease-in`: comienza lentamente, luego se acelera.
    
- `ease-out`: comienza rápido y desacelera al final.
    
- `ease-in-out`: combinación de `ease-in` y `ease-out`.
    
- `cubic-bezier(n,n,n,n)`: permite personalizar completamente la curva de aceleración (ideal para transiciones avanzadas).


A tener en cuenta:


- **No todas las propiedades son animables.** Por ejemplo, propiedades como `display` no pueden transicionarse directamente.
    
- Las transiciones se activan con **cambios de estado**, como `:hover`, `:focus`, o mediante JavaScript que modifica estilos.
    
- **Rendimiento:** Transicionar propiedades como `width`, `height`, y especialmente `box-shadow` o `filter`, puede impactar el rendimiento si se abusa. Se recomienda usar `transform` y `opacity` cuando sea posible.
