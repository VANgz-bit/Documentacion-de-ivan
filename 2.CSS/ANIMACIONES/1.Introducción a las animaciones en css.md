
En el desarrollo web moderno, las animaciones juegan un papel clave para mejorar la experiencia del usuario. Permiten que los sitios sean más interactivos, visualmente atractivos y dinámicos. Ya sea para hacer que un botón resalte, que un menú se despliegue suavemente o que un elemento aparezca de forma elegante en la pantalla, las animaciones en CSS ofrecen una manera sencilla y eficiente de lograr estos efectos sin necesidad de JavaScript.


### ¿Qué son las animaciones en CSS?

Las **animaciones en CSS** son efectos visuales que permiten cambiar de forma gradual los estilos de un elemento HTML a lo largo del tiempo. A través de ellas, se pueden modificar propiedades como el color, tamaño, posición, opacidad, rotación, entre otras, haciendo que los cambios ocurran de manera fluida en lugar de ser instantáneos.

CSS ofrece dos formas principales de animar elementos:

1. **Transiciones (`transition`)**: para cambios simples entre dos estados (por ejemplo, al pasar el mouse).
    
2. **Animaciones (`@keyframes` + `animation`)**: para crear secuencias más complejas y personalizadas, sin necesidad de interacción del usuario.

#### **Diferencia entre `transition` y `animation`**

- `transition`: requiere un cambio en el estado (como un `:hover`) y es más limitada.
    
- `animation`: puede ejecutarse automáticamente, repetirse, tener múltiples etapas y no necesita interacción del usuario.


#### **Compatibilidad**

Las animaciones CSS son compatibles con todos los navegadores modernos. Sin embargo, siempre es buena práctica probar en varios para asegurarte de que se ve bien.
