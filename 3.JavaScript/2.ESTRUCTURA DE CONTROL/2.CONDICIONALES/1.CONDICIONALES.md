## ¿Qué es una condicional?

Una **condicional** es una estructura de control que permite tomar decisiones dentro de un programa.  
Su función principal es **ejecutar diferentes bloques de código según si una condición resulta verdadera (`true`) o falsa (`false`)**.

En otras palabras, permiten que el flujo del programa cambie dependiendo de determinadas circunstancias.

---

## `if` (si)

El `if` es la estructura condicional más básica.  
Se ejecuta **solo si la condición evaluada es verdadera**.

### Sintaxis

```js
if (condición) {
  // Código que se ejecuta si la condición es verdadera
}
```

### Ejemplo

```js
let edad = 20;

if (edad >= 18) {
  console.log("Es mayor de edad");
}
```

---

## `else` (sino)

Se utiliza junto con un `if`.  
Si la condición del `if` resulta falsa, entonces se ejecuta el bloque de código dentro del `else`.

### Sintaxis

```js
if (condición) {
  // Si la condición es verdadera
} else {
  // Si la condición es falsa
}
```

### Ejemplo

```js
let tieneEntrada = false;

if (tieneEntrada) {
  console.log("Puede entrar");
} else {
  console.log("No puede entrar");
}
```

---

## `else if` (sino si)

Se emplea cuando se desean **evaluar múltiples condiciones**.  
Primero se analiza la condición del `if`; si esta no se cumple, se prueban las condiciones de los `else if`.  
Si ninguna se cumple, se ejecuta el bloque final `else` (opcional).

### Sintaxis

```js
if (condición1) {
  // Si se cumple condición1
}  (condición2) {
  // Si se cumple condición2
} else {
  // Si no se cumple ninguna
}
```

### Ejemplo

```js
let nota = 7;

if (nota >= 9) {
  console.log("Excelente");
} else if (nota >= 6) {
  console.log("Aprobado");
} else {
  console.log("Reprobado");
}
```

---

## Valores Falsy y Truthy en una condicional

En JavaScript, no siempre se evalúan condiciones estrictamente como `true` o `false`.  
Algunos valores se consideran automáticamente **falsos** (`Falsy`), mientras que todo lo demás se considera **verdadero** (`Truthy`).

### Valores Falsy:

- `false`
    
- `0`
    
- `""` (cadena vacía)
    
- `null`
    
- `undefined`
    
- `NaN`
    

### Ejemplo

```js
let edad = 22;
let tieneID = true;

if (edad >= 18 && tieneID) {
  console.log("Puede entrar al bar");
}
```

## Operador ternario (`condición ? valorSiTrue : valorSiFalse`)

El **operador ternario** es una forma compacta de escribir una condición.  
Su nombre proviene de que utiliza **tres partes**:

1. **Condición a evaluar**
    
2. **Resultado si es verdadera**
    
3. **Resultado si es falsa**
    

### Sintaxis

```js
condición ? expresión_si_true : expresión_si_false
```

### Ejemplo básico

```js
let edad = 18;
let mensaje = (edad >= 18) ? "Es mayor de edad" : "Es menor de edad";

console.log(mensaje);
```

- Si la condición `(edad >= 18)` es verdadera → `mensaje` recibe `"Es mayor de edad"`.
    
- Si es falsa → `mensaje` recibe `"Es menor de edad"`.
    

### Ventajas del operador ternario

- Permite escribir condiciones simples en **una sola línea**.
    
- Es útil cuando se desea **asignar valores rápidamente** según una condición.
    

### Ejemplo práctico

```js
let usuarioLogueado = true;

// En lugar de escribir un if/else
let mensaje = usuarioLogueado 
  ? "Bienvenido de nuevo" 
  : "Por favor, inicia sesión";

console.log(mensaje);
```

### Anidación de ternarios (con precaución)

Se pueden encadenar múltiples ternarios, aunque es recomendable no abusar de esto porque puede afectar la legibilidad del código.

```js
let nota = 8;

let resultado = (nota >= 9) 
  ? "Excelente" 
  : (nota >= 6) 
    ? "Aprobado" 
    : "Reprobado";

console.log(resultado);
```

---

## Ejemplo completo con `if/else` y ternario

```js
let nombre = "Juan";
let edad = 16;

// Usando if/else
if (edad >= 18) {
  console.log(nombre + " puede votar");
} else {
  console.log(nombre + " NO puede votar");
}

// Usando operador ternario
let mensaje = (edad >= 18) 
  ? nombre + " puede votar" 
  : nombre + " NO puede votar";

console.log(mensaje);
```

---

## Resumen rápido

|Estructura|¿Cuándo se usa?|
|---|---|
|`if`|Cuando se cumple una condición simple|
|`else`|Cuando la condición inicial es falsa|
|`else if`|Cuando hay múltiples condiciones a evaluar|
|`? :` (ternario)|Para condiciones cortas y asignación rápida de valores|
