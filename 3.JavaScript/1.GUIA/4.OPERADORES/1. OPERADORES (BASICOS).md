# 📘 Operadores en JavaScript

En JavaScript, los **operadores** son símbolos que permiten realizar operaciones sobre **valores, variables o expresiones**.  
Se utilizan para **asignar valores**, **compararlos**, **realizar cálculos matemáticos**, entre muchas otras acciones.

Ejemplo básico:

```js
let suma = 5 + 3; 
// "+" es un operador aritmético que suma dos números
```

---

## 🔹 Operadores de Asignación

### ¿Qué son?

Los **operadores de asignación** permiten **dar un valor inicial a una variable o actualizarla**.  
El más usado es el **signo igual (`=`)**, pero existen versiones combinadas con operadores aritméticos que simplifican el código.

---

### 📑 Lista de operadores de asignación

|Operador|Nombre|Ejemplo|Equivalente a...|Descripción|
|---|---|---|---|---|
|`=`|Asignación|`x = 10`|`x = 10`|Asigna un valor a la variable.|
|`+=`|Asignación con suma|`x += 5`|`x = x + 5`|Suma y reasigna el valor.|
|`-=`|Asignación con resta|`x -= 2`|`x = x - 2`|Resta y reasigna el valor.|
|`*=`|Asignación con multiplicación|`x *= 3`|`x = x * 3`|Multiplica y reasigna el valor.|
|`/=`|Asignación con división|`x /= 2`|`x = x / 2`|Divide y reasigna el valor.|
|`%=`|Asignación con módulo|`x %= 3`|`x = x % 3`|Guarda el resto de una división.|
|`**=`|Asignación con potencia|`x **= 2`|`x = x ** 2`|Eleva y reasigna el valor.|

---

### 📌 Ejemplo práctico

```js
let x = 10;

x += 5;  // 15
x -= 2;  // 13
x *= 2;  // 26
x /= 2;  // 13
x %= 4;  // 1
x **= 3; // 1 → porque 1³ = 1
```

## 🔹 Operadores Aritméticos

### ¿Qué son?

Los **operadores aritméticos** se usan para realizar cálculos matemáticos.  
Funcionan de forma similar a los que usamos en matemáticas.

---

### 📑 Lista de operadores aritméticos

| Operador | Nombre               | Ejemplo       | Resultado       | Descripción                                         |
| -------- | -------------------- | ------------- | --------------- | --------------------------------------------------- |
| `+`      | Suma                 | `5 + 3`       | `8`             | Suma dos valores. También concatena strings.        |
| `-`      | Resta                | `10 - 4`      | `6`             | Resta el segundo valor al primero.                  |
| `*`      | Multiplicación       | `2 * 3`       | `6`             | Multiplica dos valores.                             |
| `/`      | División             | `10 / 2`      | `5`             | Divide el primer valor por el segundo.              |
| `%`      | Módulo (resto)       | `10 % 3`      | `1`             | Devuelve el resto de la división.                   |
| `**`     | Potencia (exponente) | `2 ** 3`      | `8`             | Eleva el primer número al exponente del segundo.    |
| `++`     | Incremento           | `x++` / `++x` | Depende del uso | Suma 1 a la variable. Puede ser posfijo o prefijo.  |
| `--`     | Decremento           | `x--` / `--x` | Depende del uso | Resta 1 a la variable. Puede ser posfijo o prefijo. |

---

### 📌 Ejemplo práctico

```js
let a = 5;
let b = 2;

console.log(a + b);  // 7
console.log(a - b);  // 3
console.log(a * b);  // 10
console.log(a / b);  // 2.5
console.log(a % b);  // 1
console.log(a ** b); // 25
```

---

### 🔎 Incremento y Decremento: prefijo vs posfijo

El comportamiento de `++` y `--` cambia según se use **antes** o **después** de la variable:

```js
let x = 5;

console.log(x++); // 5 → primero devuelve el valor, luego incrementa (posfijo)
console.log(x);   // 6

let y = 5;

console.log(++y); // 6 → primero incrementa, luego devuelve el valor (prefijo)
```

---

### ⚠️ Precedencia de operadores

En expresiones complejas, algunos operadores se ejecutan antes que otros.  
El orden general es:

1. Paréntesis `()`
    
2. Potencia `**`
    
3. Multiplicación, división, módulo `* / %`
    
4. Suma y resta `+ -`
    

```js
console.log(2 + 3 * 4);  // 14 (multiplicación antes que suma)
console.log((2 + 3) * 4); // 20 (los paréntesis tienen prioridad)
```

---

### 📌 Concatenación con `+`

El operador `+` también sirve para **unir strings**.

```js
let nombre = "Ana";
let saludo = "Hola " + nombre;

console.log(saludo); // "Hola Ana"
```

⚠️ Si uno de los operandos es un string, JavaScript convierte el otro a string automáticamente (coerción implícita).

```js
console.log(5 + "2"); // "52"
```

---

## 🔹 Comparativa: Asignación vs Aritméticos

|Característica|Operadores de Asignación|Operadores Aritméticos|
|---|---|---|
|Propósito principal|Asignar o actualizar valores|Realizar cálculos matemáticos|
|Modifican la variable original|✅ Sí|❌ No necesariamente|
|Uso común|`x += 2`, `x *= 3`|`a + b`, `a * b`|
|Se combinan con otros|Sí, con aritméticos|Sí, en expresiones matemáticas|
|Actúan sobre|Una variable y un valor|Dos valores (variables o literales)|
|Tareas típicas|Actualización de datos|Cálculos y transformaciones|

---

## 📌 Resumen final

- 🔹 **Operadores de asignación** → modifican directamente el valor de una variable.  
    Ej: `x += 2` suma 2 a `x`.
    
- 🔹 **Operadores aritméticos** → realizan cálculos, pero no modifican los valores originales salvo que se combinen con asignación.  
    Ej: `5 + 3` da `8`, pero no cambia `5` ni `3`.
