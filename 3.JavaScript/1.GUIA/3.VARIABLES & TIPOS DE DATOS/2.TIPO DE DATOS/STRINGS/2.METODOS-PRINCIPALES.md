# 📘 Métodos de String en JavaScript (Principales)

---
## 1. Definición

Los **métodos de `String`** son **funciones integradas** en JavaScript que permiten **manipular, analizar y transformar cadenas de texto**.  
Un `String` es un tipo primitivo, pero cuando se accede a un método, JavaScript lo convierte temporalmente en un objeto `String` (autoboxing) para ejecutar la operación.

👉 Existen más de 30 métodos, pero los más usados se agrupan en:

1. **Obtención de información** (`.length`, `.charAt()`, `.indexOf()`, etc.)
    
2. **Transformación de texto** (`.toUpperCase()`, `.toLowerCase()`, `.trim()`, etc.)
    
3. **Extracción de partes** (`.slice()`, `.substring()`, `.substr()` [obsoleto])
    
4. **Búsqueda y coincidencias** (`.includes()`, `.startsWith()`, `.endsWith()`, `.match()`)
    
5. **Reemplazo y división** (`.replace()`, `.replaceAll()`, `.split()`)
    

---

## 2. Principales métodos (con explicación completa)

### 🔹 `.length`

Devuelve la **cantidad de caracteres** en el string.

```js
let texto = "Hola Mundo";
console.log(texto.length); // 10
```

⚠️ Curiosidad: algunos emojis cuentan como 2 unidades.

---

### 🔹 `.toUpperCase()` y `.toLowerCase()`

Transforman todo el texto a **mayúsculas** o **minúsculas**.

```js
let palabra = "JavaScript";
console.log(palabra.toUpperCase()); // "JAVASCRIPT"
console.log(palabra.toLowerCase()); // "javascript"
```

---

### 🔹 `.trim()`, `.trimStart()`, `.trimEnd()`

Eliminan espacios en blanco al inicio y/o final.

```js

> let texto = "   Hola   ";

console.log(texto.trim());      // "Hola"
console.log(texto.trimStart()); // "Hola   "
console.log(texto.trimEnd());   // "   Hola"
```

---

### 🔹 `.charAt()` y acceso con corchetes `[]`

Obtienen un carácter en una posición específica.

```js
let palabra = "Hola";
console.log(palabra.charAt(1)); // "o"
console.log(palabra[1]);        // "o"
```

---

### 🔹 `.indexOf()` y `.lastIndexOf()`

Buscan la posición de la **primera/última aparición** de un substring.

```js
let texto = "banana";
console.log(texto.indexOf("a"));      // 1
console.log(texto.lastIndexOf("a"));  // 5
```

❌ Si no encuentra nada → devuelve `-1`.

---

### 🔹 `.includes()`

Devuelve `true` o `false` si el substring existe.

```js
console.log("JavaScript".includes("Script")); // true
console.log("JavaScript".includes("Java", 5)); // false (empieza a buscar desde índice 5)
```

---

### 🔹 `.startsWith()` y `.endsWith()`

Verifican si un string comienza o termina con cierta secuencia.

```js
let archivo = "informe.pdf";
console.log(archivo.startsWith("info")); // true
console.log(archivo.endsWith(".pdf"));   // true
```

---

### 🔹 `.slice()`

Extrae una parte del string con inicio y fin opcional.

```js
let texto = "JavaScript";
console.log(texto.slice(0, 4));  // "Java"
console.log(texto.slice(-6));    // "Script"
```

⚠️ Admite índices negativos.

---

### 🔹 `.substring()`

Similar a `.slice()` pero **no acepta negativos**.

```js
let texto = "JavaScript";
console.log(texto.substring(0, 4)); // "Java"
console.log(texto.substring(4, 0)); // "Java" (intercambia valores)
```

---

### 🔹 `.replace()` y `.replaceAll()`

Reemplazan coincidencias.

```js
let texto = "hola mundo mundo";
console.log(texto.replace("mundo", "JS"));   // "hola JS mundo"
console.log(texto.replaceAll("mundo", "JS"));// "hola JS JS"
```

⚠️ `.replaceAll()` es más moderno (ES2021).

---

### 🔹 `.split()`

Convierte un string en array, usando un separador.

```js
let texto = "rojo,verde,azul";
console.log(texto.split(",")); // ["rojo", "verde", "azul"]
```

---

### 🔹 `.match()` y `.matchAll()`

Trabajan con expresiones regulares para obtener coincidencias.

```js
let texto = "abc abc abc";
console.log(texto.match(/abc/g)); 
// ["abc", "abc", "abc"]

for (let match of texto.matchAll(/abc/g)) {
  console.log(match[0]); 
}
```

---

## 3. Características

- Casi todos los métodos devuelven **un nuevo string** (no modifican el original).
    
- Algunos métodos (`.length`) devuelven **números**.
    
- `.replace()` y `.split()` permiten usar **expresiones regulares** para mayor potencia.
    
- `.substring()` y `.slice()` parecen iguales, pero difieren en cómo manejan índices negativos.
    

---

## 4. Ejemplos buenos y malos

✔️ Buenos:

```js
let nombre = " Iván ";
console.log(nombre.trim().toUpperCase()); // "IVÁN"
```

❌ Malos:

```js
let texto = "Hola";
texto[0] = "h"; // ❌ inmutable, no cambia
```

---

## 5. Casos especiales

- `.length` puede contar mal algunos caracteres Unicode (ej. emojis → ocupan 2 unidades).
    
- `.replace()` solo reemplaza la **primera coincidencia** a menos que uses regex global o `.replaceAll()`.
    
- `.substring()` cambia el orden de los índices si el primero es mayor que el segundo.
    

---

## 6. Buenas prácticas

- Usar **template literals** en vez de concatenaciones con `+`.
    
- Usar `.includes()` en lugar de `.indexOf() >= 0` (más legible).
    
- Evitar `.substring()`, preferir `.slice()` (más consistente).
    
- Para reemplazos múltiples → usar `.replaceAll()` si es posible.
    

---

## 7. Resumen

- Los métodos de `String` permiten **consultar, transformar y analizar texto**.
    
- Se agrupan en **búsqueda, extracción, transformación y reemplazo**.
    
- Los strings son **inmutables** → los métodos devuelven **nuevos valores**.
    
- Algunos métodos modernos (`replaceAll`, `trimStart`, `trimEnd`) están disponibles desde ES2020+.
    

---

📌 **Relación con otros temas:**

- Se conecta con **arrays** (ej: `.split()` devuelve un array).
    
- Relacionado con **expresiones regulares** (`.match()`, `.replace()`).
    
- Muy usado junto con **operadores de concatenación** (`+`, template literals).
