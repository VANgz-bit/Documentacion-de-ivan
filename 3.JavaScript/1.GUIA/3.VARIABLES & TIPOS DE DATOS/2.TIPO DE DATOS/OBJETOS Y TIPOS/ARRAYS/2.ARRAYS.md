# 📘 `Array` en JavaScript (Objeto General)

## 🔹 Definición

Un **Array** en JavaScript es un **objeto especial** diseñado para **almacenar colecciones de datos ordenados**.  
Cada valor dentro de un array se llama **elemento**, y cada elemento tiene una **posición numérica llamada índice**, que siempre comienza en `0`.

Aunque los arrays son técnicamente objetos (`typeof [] === "object"`), tienen un **comportamiento particular** que los diferencia de los objetos regulares:

- Mantienen el **orden de los elementos**.
    
- Tienen la propiedad **`length`** que refleja la cantidad de elementos.
    
- Están optimizados internamente para operaciones de acceso y recorrido secuencial.
    

---

## 🔹 Características principales

1. **Colección indexada**: cada elemento se accede por índice (`array[0]`, `array[1]`, etc.).
    
2. **Heterogeneidad**: pueden almacenar cualquier tipo de dato (números, strings, objetos, incluso otros arrays).
    
3. **Mutabilidad**: sus valores pueden modificarse después de creados.
    
4. **Dinámicos**: su tamaño puede crecer o reducirse automáticamente al agregar o eliminar elementos.
    
5. **Herencia prototípica**: todos los arrays heredan propiedades y métodos de `Array.prototype`.
    

---

## 🔹 Creación de Arrays

Existen varias formas de crear un array en JavaScript:

### 1. **Usando un literal de array (recomendado)**

```js
let frutas = ["manzana", "banana", "pera"];
```

### 2. **Usando el constructor `Array()`**

```js
let numeros = new Array(1, 2, 3, 4);
```

⚠️ Cuidado: `new Array(3)` crea un array vacío de **longitud 3** (con huecos), no con el número 3.

### 3. **Usando `Array.of()`**

```js
let unico = Array.of(3); 
// crea [3], no un array vacío de longitud 3
```

### 4. **Usando `Array.from()`**

Convierte estructuras iterables o parecidas a arrays en un array.

```js
let cadena = Array.from("hola"); 
console.log(cadena); // ["h", "o", "l", "a"]
```

---

## 🔹 Propiedades importantes

### 1. **`length`**

- Representa la cantidad de elementos en el array.
    
- Puede modificarse para truncar o extender el array.
    

```js
let letras = ["a", "b", "c"];
console.log(letras.length); // 3
letras.length = 2; 
console.log(letras); // ["a", "b"]
```

---

## 🔹 Métodos principales (explicados)

### 1. **Mutadores (modifican el array original)**

- **`push()`** → agrega uno o más elementos al final.
    

```js
let arr = [1, 2];
arr.push(numeral);
console.log(arr); // [1, 2, 3]
```

- **`pop()`** → elimina y devuelve el último elemento.
    

```js
let arr = [1, 2, 3];
let ultimo = arr.pop();
console.log(ultimo); // 3
console.log(arr); // [1, 2]
```

- **`shift()`** → elimina y devuelve el primer elemento.
    

```js
let arr = [1, 2, 3];
let primero = arr.shift();
console.log(primero); // 1
console.log(arr); // [2, 3]
```

- **`unshift()`** → agrega elementos al inicio.
    

```js
let arr = [2, 3];
arr.unshift(1);
console.log(arr); // [1, 2, 3]
```

- **`splice()`** → agrega, elimina o reemplaza elementos.
    

```js
let arr = ["a", "b", "c"];
arr.splice(1, 1, "z"); 
console.log(arr); // ["a", "z", "c"]
```

- **`reverse()`** → invierte el orden.
    

```js
let arr = [1, 2, 3];
arr.reverse();
console.log(arr); // [3, 2, 1]
```

- **`sort()`** → ordena los elementos (por defecto alfabéticamente).
    

```js
let numeros = [10, 2, 5];
numeros.sort(); 
console.log(numeros); // [10, 2, 5] (por string)
numeros.sort((a, b) => a - b); 
console.log(numeros); // [2, 5, 10]
```

---

### 2. **No mutadores (no alteran el array original)**

- **`slice()`** → devuelve una copia parcial.
    

```js
let arr = ["a", "b", "c"];
console.log(arr.slice(0, 2)); // ["a", "b"]
```

- **`concat()`** → une arrays.
    

```js
let arr1 = [1, 2];
let arr2 = [3, 4];
console.log(arr1.concat(arr2)); // [1, 2, 3, 4]
```

- **`join()`** → convierte los elementos en un string.
    

```js
let arr = ["h", "o", "l", "a"];
console.log(arr.join("-")); // "h-o-l-a"
```

- **`indexOf()` / `lastIndexOf()`** → buscan la posición de un elemento.
    

```js
let arr = ["a", "b", "a"];
console.log(arr.indexOf("a")); // 0
console.log(arr.lastIndexOf("a")); // 2
```

- **`includes()`** → verifica si un elemento existe.
    

```js
let arr = [1, 2, 3];
console.log(arr.includes(2)); // true
```

---

### 3. **De recorrido / transformación**

- **`forEach()`** → ejecuta una función por cada elemento.
    

```js
["a", "b", "c"].forEach(el => console.log(el));
```

- **`map()`** → crea un nuevo array con resultados de aplicar una función.
    

```js
let numeros = [1, 2, 3];
let dobles = numeros.map(n => n * 2);
console.log(dobles); // [2, 4, 6]
```

- **`filter()`** → devuelve un nuevo array con elementos que cumplen condición.
    

```js
let numeros = [1, 2, 3, 4];
let pares = numeros.filter(n => n % 2 === 0);
console.log(pares); // [2, 4]
```

- **`reduce()`** → acumula valores en un solo resultado.
    

```js
let numeros = [1, 2, 3, 4];
let suma = numeros.reduce((acc, n) => acc + n, 0);
console.log(suma); // 10
```

- **`find()` / `findIndex()`** → busca el primer elemento que cumple condición.
    

```js
let numeros = [10, 20, 30];
console.log(numeros.find(n => n > 15)); // 20
console.log(numeros.findIndex(n => n > 15)); // 1
```

- **`some()` / `every()`** → verifican condiciones.
    

```js
let numeros = [1, 2, 3];
console.log(numeros.some(n => n > 2)); // true
console.log(numeros.every(n => n > 0)); // true
```

---

## 🔹 Conclusión

El objeto **`Array`** es una de las estructuras de datos más utilizadas en JavaScript.  
Permite **almacenar, modificar, recorrer y transformar colecciones de valores** mediante una amplia variedad de **métodos y propiedades**.  
Los arrays son **objetos especializados**, y la sintaxis de **array literal `[]`** es simplemente la forma más práctica de crearlos.
